## 🔥 Breaking

### DeepSeek V4 (Flash & Pro) — sortie aujourd'hui
- **Quoi** : DeepSeek publie en preview ses deux nouveaux modèles flagship sous licence MIT : V4 Flash (284B total / 13B actifs) et V4 Pro (1,6T total / 49B actifs), avec 1 million de tokens de contexte et une nouvelle architecture Hybrid Attention
- **Lien** : https://www.bloomberg.com/news/articles/2026-04-24/deepseek-unveils-newest-flagship-a-year-after-ai-breakthrough — https://www.cnbc.com/2026/04/24/deepseek-v4-llm-preview-open-source-ai-competition-china.html
- **Pourquoi c'est important** : Performances proches de GPT-5.4 et Gemini 3.1 Pro à une fraction du prix ($0,14/M tokens input pour Flash, $1,74/M pour Pro). Open source MIT. L'effet DeepSeek-R1 se répète : pression massive sur les marges des modèles occidentaux
- **Testable** : Oui — disponible en preview via l'API DeepSeek dès aujourd'hui

### GPT-5.5 — déployé aux abonnés payants
- **Quoi** : OpenAI déploie GPT-5.5 pour Plus, Pro, Business et Enterprise dans ChatGPT et Codex ; conçu pour les tâches agentiques longues (code, recherche web, opération d'outils, données)
- **Lien** : https://www.cnbc.com/2026/04/23/openai-announces-latest-intelligence-model.html
- **Pourquoi c'est important** : Contexte 1 million de tokens, 85% sur GDPval vs human workers, meilleure navigation dans l'ambiguïté des tâches multi-étapes. Positionné comme le rival direct de Claude Opus 4.7 pour les agents autonomes
- **Testable** : Oui — abonnés ChatGPT Plus et supérieur, et dans Codex

- **Quoi** : Anthropic ouvre en beta publique la mémoire persistante pour ses agents managés : les agents apprennent des sessions passées et partagent leur mémoire entre eux

- **Testable** : Oui — beta publique accessible via l'API Anthropic

### Google Colab MCP Server — officiel et gratuit
- **Quoi** : Google publie un MCP server officiel pour Colab permettant à des agents IA de créer des notebooks, exécuter du code, gérer des dépendances et retourner les résultats
- **Lien** : https://www.epsilla.com/blogs/ai-agents-sandbox-mcp-april-2026
- **Pourquoi c'est important** : Ouvre Colab comme sandbox d'exécution gratuit pour agents — use case évident : agents de data science et d'analyse en Python pilotés par LLM
- **Testable** : Oui — gratuit, disponible sur Colab

### Microsoft Power Apps MCP Server
- **Quoi** : Microsoft intègre un MCP server dans Power Apps avec une file d'approbation humaine, permettant de connecter des agents IA à 1 100 systèmes d'entreprise sans code

### xAI — APIs Grok STT & TTS en production
- **Quoi** : xAI lance deux APIs audio standalone : Speech-to-Text (timestamps au mot, diarisation, 25+ langues) et Text-to-Speech, issues de la même stack que Grok Voice, Tesla et Starlink
