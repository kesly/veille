### Cloudflare Code Mode MCP Server
- **Quoi** : MCP server Cloudflare exposant 2 500+ endpoints API en seulement ~1 000 tokens via deux outils : `search()` et `execute()`
- **Lien** : https://blog.cloudflare.com/code-mode-mcp/
- **Pourquoi c'est important** : Réduction de 81 % de consommation de tokens vs un MCP classique. Le modèle génère et exécute du JavaScript dans un isolate V8 sécurisé. OAuth 2.1 natif. Les nouveaux endpoints Cloudflare sont automatiquement accessibles sans redéploiement

- **Pourquoi c'est important** : Alerte critique pour tous les pipelines de synthetic data augmentation. Le risque de propagation de biais dans les modèles de seconde génération est documenté empiriquement
