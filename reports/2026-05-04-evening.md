# AI Watch — 2026-05-04 Soir

## 🔥 Breaking
> Releases majeures, annonces qui changent la donne

### Claude Mythos Preview — Anthropic
- **Quoi** : Modèle IA spécialisé cybersécurité capable de trouver des zero-days dans les principaux OS et navigateurs, avec 83 %+ de taux de succès d'exploitation au premier essai
- **Lien** : https://red.anthropic.com/2026/mythos-preview/
- **Pourquoi c'est important** : Premier modèle à réussir une simulation d'attaque réseau d'entreprise en 32 étapes de bout en bout. Release restreinte (pas d'accès public) — Anthropic lance Project Glasswing, un consortium (Apple, Microsoft, Amazon, Cisco, CrowdStrike, Linux Foundation…) pour patcher les vulnérabilités détectées. Redéfinit la frontière entre outil défensif et offensif.
- **Testable** : Non — accès uniquement pour ~40 organisations critiques sélectionnées

### OpenAI Workspace Agents — ChatGPT
- **Quoi** : Agents Codex embarqués dans ChatGPT Enterprise/Business, capables de s'intégrer à Slack, Salesforce, GitHub, de s'exécuter en cloud sur planification ou déclencheur
- **Lien** : https://openai.com/index/introducing-workspace-agents-in-chatgpt/
- **Pourquoi c'est important** : Successeur direct des GPTs, avec exécution de workflows longs en arrière-plan. Devient payant (crédits) à partir du **6 mai 2026**. Signal fort qu'OpenAI cible l'entreprise comme terrain de monétisation principal.
- **Testable** : Oui — gratuit jusqu'au 6 mai sur ChatGPT Business/Enterprise/Edu

### xAI Grok 4.3 + Custom Voices
- **Quoi** : Nouveau modèle Grok 4.3 (1M tokens de contexte, prix API agressif) + clonage vocal en < 2 minutes à partir d'une minute de parole
- **Lien** : https://venturebeat.com/technology/xai-launches-grok-4-3-at-an-aggressively-low-price-and-a-new-fast-powerful-voice-cloning-suite
- **Pourquoi c'est important** : Premier acteur frontier à proposer le voice cloning dans l'API de base, gratuit sur la console xAI. Sécurité intégrée (passphrase + speaker embedding). Contexte 1M tokens rivalise directement avec Gemini.
- **Testable** : Oui — console xAI, gratuit (https://x.ai/news/grok-custom-voices)

## 🆕 Nouveaux Outils & Produits

### Craft Agents OSS — Lukilabs
- **Quoi** : Framework open-source (Apache 2.0) de gestion d'agents IA avec inbox multi-session, support multi-LLM (Anthropic, OpenAI, Google, GitHub Copilot), visualisation des outils en temps réel
- **Lien** : https://github.com/lukilabs/craft-agents-oss
- **Pourquoi c'est important** : Trending GitHub dès sa sortie le 2 mai. Multi-provider natif, idéal pour gérer plusieurs agents sur plusieurs sessions sans dépendance à un seul fournisseur.
- **Testable** : Oui — open source, installation locale (Apache 2.0, gratuit)

### Mistral Workflows (Public Preview)
- **Quoi** : Orchestration d'agents durable et observable dans Le Chat/Studio : approbations human-in-the-loop, exécution traçable, déploiement cloud/on-prem/hybride
- **Lien** : https://mistral.ai/news
- **Pourquoi c'est important** : Mistral entre dans le segment "enterprise agentic orchestration" en concurrence directe avec LangGraph et Temporal. Le déploiement on-prem est un différenciateur fort pour les entreprises européennes soucieuses de souveraineté.
- **Testable** : Oui — public preview sur mistral.ai (freemium)

### GitHub Copilot CLI — mise à jour majeure (2 mai)
- **Quoi** : Meilleur streaming, gestion améliorée des agents et sessions, support MCP et OAuth renforcé, démarrage plus rapide
- **Lien** : https://releasebot.io/updates/categories/ai-coding-assistants
- **Pourquoi c'est important** : Copilot CLI devient un vrai agent de terminal avec MCP natif — aligne GitHub sur Claude Code et Gemini CLI qui proposent déjà cette expérience.
- **Testable** : Oui — mise à jour via `npm install -g @githubnext/github-copilot-cli` (payant, abonnement Copilot)

### Gemini Workspace Intelligence
- **Quoi** : Gemini reçoit un contexte temps réel depuis Gmail, Chat, Agenda et Drive pour les tâches génératives sans re-prompting manuel
- **Lien** : https://releasebot.io/updates/google/gemini
- **Pourquoi c'est important** : Transforme Gemini en "couche de travail permanente" plutôt qu'assistant ponctuel. Intégration automobile également : Gemini arrive dans les véhicules GM via mise à jour OTA.
- **Testable** : Oui — disponible progressivement dans Google Workspace (abonnement Business/Enterprise)
