## 🆕 Nouveaux Outils & Produits

### Claude Design (Anthropic, 17 avril)
- **Quoi** : Outil de création visuelle (prototypes, slides, mockups, one-pagers) propulsé par Claude Opus 4.7
- **Lien** : https://techcrunch.com/2026/04/17/anthropic-launches-claude-design-a-new-product-for-creating-quick-visuals/
- **Pourquoi c'est important** : A fait chuter l'action Adobe de -1,5 % et Figma de -7 % sur les marchés secondaires dès l'annonce. Lit le codebase et les design files pour appliquer automatiquement le design system de l'entreprise
- **Testable** : Oui — research preview pour abonnés Pro, Max, Team, Enterprise

### Claude Code Desktop Redesign + Routines (Anthropic, 14 avril)
- **Quoi** : Refonte complète de l'appli desktop Claude Code + lancement des "Routines" (automatisations cloud planifiées)
- **Lien** : https://siliconangle.com/2026/04/14/anthropics-claude-code-gets-automated-routines-desktop-makeover/
- **Pourquoi c'est important** : Les Routines (Scheduled / API / Webhook) s'exécutent sur l'infra Anthropic même laptop éteint. Intégration native terminal + diff viewer + file editor. Pro : 5 routines/jour, Max : 15, Team/Enterprise : 25
- **Testable** : Oui — disponible pour tous les abonnés Claude Code (Pro+)

### xAI Grok STT & TTS APIs (18 avril)
- **Quoi** : APIs voix standalone de xAI — transcription 25 langues (STT) + synthèse 5 voix expressives / 20 langues (TTS) à 4,20 $/M caractères
- **Lien** : https://www.marktechpost.com/2026/04/18/xai-launches-standalone-grok-speech-to-text-and-text-to-speech-apis-targeting-enterprise-voice-developers/
- **Pourquoi c'est important** : Même infra que Grok Voice sur mobile, Tesla et Starlink. Positionnement direct face à ElevenLabs et OpenAI Audio
- **Testable** : Oui — API disponible (STT en GA, TTS en early access)

### Google Gemini 3.1 Flash TTS (15 avril)
- **Quoi** : Moteur de synthèse vocale expressif avec "audio tags" — commandes en langage naturel directement dans le texte via syntaxe `[...]`
- **Lien** : https://www.buildfastwithai.com/blogs/gemini-3-1-flash-tts-google-ai-voice-model-2026
- **Pourquoi c'est important** : Contrôle fin du ton, débit et émotion sans paramètres séparés — différenciateur fort face aux TTS classiques
- **Testable** : Oui — Gemini API et Google AI Studio

### Microsoft Agent Framework 1.0 GA (3 avril)
- **Quoi** : SDK unifié .NET + Python fusionnant Semantic Kernel et AutoGen, avec support natif MCP et A2A
- **Lien** : https://www.digitalapplied.com/blog/microsoft-agent-framework-1-0-dotnet-python-guide
- **Pourquoi c'est important** : Standard stable pour agents enterprise multi-framework. 70 % des SaaS majeurs proposent déjà des MCP servers ; MCP v2 intègre OAuth 2.1 et SSO
- **Testable** : Oui — open source, packages NuGet et PyPI

### Voicebox — workstation synthèse vocale open source
- **Quoi** : Environnement complet de synthèse vocale (par jamiepine), trending GitHub
- **Lien** : https://aitoolly.com/ai-news/article/2026-04-19-voicebox-a-new-open-source-speech-synthesis-workstation-emerges-on-github
- **Pourquoi c'est important** : Alternative open source à ElevenLabs pour un workflow complet de TTS local
- **Testable** : Oui — open source, gratuit, GitHub
