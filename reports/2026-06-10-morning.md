# AI Watch — 2026-06-10 Matin

## 🔥 Breaking

> Releases majeures, annonces qui changent la donne

### Claude Fable 5 — Premier modèle Mythos accessible au public
- **Quoi** : Anthropic a lancé le 9 juin 2026 Claude Fable 5, version publique de son modèle frontier Mythos 5, qui était jusqu'ici réservé à un cercle restreint (Project Glasswing).
- **Lien** : https://techcrunch.com/2026/06/09/anthropic-released-claude-fable-5-its-most-powerful-model-publicly-days-after-warning-ai-is-getting-too-dangerous/
- **Pourquoi c'est important** : Fable 5 est classé n°1 sur quasi tous les benchmarks (software engineering, vision, recherche scientifique). À $10/M tokens en input et $50/M en output — le double d'Opus 4.8 — il cible les usages professionnels exigeants. Il est intégré à GitHub Copilot dès le jour de lancement. En parallèle, Mythos 5 (avec certaines protections désactivées) reste réservé aux professionnels de la cybersécurité via Project Glasswing.
- **Testable** : Oui — via l'API Anthropic et les plans Enterprise. GitHub Copilot (déploiement progressif à ~10% des utilisateurs).

### Apple WWDC 2026 — Siri AI propulsé par Google Gemini
- **Quoi** : Apple a annoncé le 8 juin une refonte totale de Siri, désormais alimentée par Google Gemini, avec une compréhension contextuelle profonde du système et une nouvelle interface visuelle dite "Neural Expressive".
- **Lien** : https://techcrunch.com/2026/06/09/wwdc-2026-everything-announced-on-siri-ai-os-27-apple-intelligence-and-more/
- **Pourquoi c'est important** : Le partenariat Apple × Google (Gemini intégré dans iOS 27) est un signal fort : même Apple renonce à construire son propre modèle frontier et choisit Gemini plutôt qu'OpenAI. Siri devient une app dédiée avec mode caméra (voir ce que l'utilisateur voit en temps réel) et voix plus naturelles avec rythme ajustable.
- **Testable** : Non pour l'instant — bêta anglais prévue "plus tard cette année" aux États-Unis, hors UE au lancement.

## 🆕 Nouveaux Outils & Produits

### Microsoft MAI — 7 modèles in-house annoncés à Build 2026
- **Quoi** : Microsoft a lancé une famille de 7 modèles développés en interne (sans dépendance à OpenAI) : MAI-Thinking-1 (raisonnement), MAI-Code-1-Flash (coding), MAI-Image-2.5 (image), MAI-Transcribe-1.5 (transcription), entre autres.
- **Lien** : https://microsoft.ai/news/building-a-hillclimbing-machine-launching-seven-new-mai-models/
- **Pourquoi c'est important** : MAI-Thinking-1 atteint 97% sur AIME 2025 et 53% sur SWE-Bench Pro — niveau Opus 4.6. MAI-Code-1-Flash (5B params seulement) atteint 51% sur SWE-Bench Pro et déploie dans GitHub Copilot CLI. Microsoft prend son indépendance technologique vis-à-vis d'OpenAI.
- **Testable** : Oui — MAI-Code-1-Flash déjà dans GitHub Copilot (10% des utilisateurs individuels).

### Google Antigravity — Remplacement de Gemini Code Assist et Gemini CLI
- **Quoi** : Google unifie ses outils développeurs IA sous une nouvelle plateforme "Antigravity". Gemini Code Assist et Gemini CLI sont dépréciés pour les utilisateurs gratuits et individuels à partir du 18 juin 2026.
- **Lien** : https://blog.logrocket.com/ai-dev-tool-power-rankings/
- **Pourquoi c'est important** : L'Antigravity CLI est écrit en Go, supporte les workflows asynchrones et remplace toute la stack de coding AI de Google en un seul outil. Transition à gérer pour tous les devs qui utilisaient Gemini Code Assist.
- **Testable** : Oui — Antigravity est disponible dès maintenant en remplacement.

### OpenCode — Coding agent open-source à 160K étoiles GitHub
- **Quoi** : OpenCode s'impose comme le coding agent open-source le plus adopté (160K+ étoiles, 7,5M utilisateurs actifs/mois), model-agnostic avec accès à 75+ providers, intégration LSP native, et licence MIT.
- **Lien** : https://usefulai.com/tools/ai-coding
- **Pourquoi c'est important** : Alternative libre et locale à Cursor/GitHub Copilot, qui tourne en air-gapped. Signe que le marché des coding agents se démocratise et sort du walled-garden.
- **Testable** : Oui — open source, installation immédiate.

### OpenAI — Mémoire "Dreaming" pour les utilisateurs gratuits
- **Quoi** : OpenAI a étendu le système de mémoire avancée ChatGPT ("Dreaming") aux utilisateurs gratuits le 8 juin 2026. Les mémoires se mettent à jour automatiquement au fil du temps.
- **Lien** : https://openai.com/news/
- **Pourquoi c'est important** : La mémoire contextuelle persistante, jusqu'ici réservée aux abonnés Pro/Plus, est maintenant universelle. Cela change profondément l'expérience ChatGPT de base.
- **Testable** : Oui — directement sur ChatGPT (gratuit).

### Noma Agent Access Control — Gouvernance des agents et serveurs MCP en entreprise
- **Quoi** : Noma a lancé un système de contrôle d'accès pour découvrir, gouverner et appliquer des politiques sur les agents IA et serveurs MCP à l'échelle de l'entreprise.
- **Lien** : https://drj.com/industry_news/noma-launches-agentic-access-control-to-govern-ai-agents-and-mcp-servers-across-the-enterprise/
- **Pourquoi c'est important** : La prolifération des agents MCP en entreprise est devenue incontrôlable — Noma répond à un vrai besoin de visibilité et de sécurité. Premier outil dédié à la gouvernance MCP.
- **Testable** : Oui — accès entreprise via noma.security.
