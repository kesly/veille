## 🆕 Nouveaux Outils & Produits

### MCP Spec 2026-07-28 — Release Candidate disponible
- **Quoi** : La plus grande révision du protocole MCP depuis son lancement — MCP devient stateless, suppression des sessions et du handshake d'initialisation
- **Lien** : https://blog.modelcontextprotocol.io/posts/2026-07-28-release-candidate/
- **Pourquoi c'est important** : Les serveurs MCP peuvent désormais tourner derrière un load balancer classique (plus de sticky sessions). Nouveau framework d'extensions (identifiants reverse-DNS), extension Tasks pour les workflows longs, MCP Apps (UI HTML embarquée dans un iframe sandboxé). Trois features dépréciées : Roots, Sampling, Logging. Spec finale le 28 juillet, bêtas SDK disponibles maintenant.
- **Testable** : Oui — bêtas SDK disponibles via https://blog.modelcontextprotocol.io/posts/sdk-betas-2026-07-28/

### Claude Fable 5 prend la tête de l'Epoch Capabilities Index
- **Quoi** : Fable 5 atteint un score de 161 sur l'ECI le 15 juin, battant GPT-5.5 Pro de 1 point
- **Lien** : https://epoch.ai/benchmarks
- **Pourquoi c'est important** : Première fois qu'Anthropic prend la tête de l'ECI en plus d'un an. L'ECI est devenu la référence principale depuis la saturation de MMLU (>88% pour tous les modèles frontières).
- **Testable** : Non

### Humanity's Last Exam — Le nouveau benchmark anti-saturation
- **Quoi** : Nouveau benchmark conçu pour résister à la saturation : meilleurs modèles à ~35%, experts humains à ~90%
- **Lien** : https://hai.stanford.edu/ai-index/2026-ai-index-report/technical-performance
- **Pourquoi c'est important** : Expose un écart de 50+ points que les anciens benchmarks ne révèlent plus. MMLU et MMLU-Pro officiellement saturés — les différences de scores en haut du classement sont statistiquement non significatives.
- **Testable** : Non

### Benchmarking Agentic Review Systems (arXiv)
- **Quoi** : Paper récent sur l'évaluation des systèmes d'agents IA pour la revue de code et de recherche
- **Lien** : https://arxiv.org/pdf/2606.19749
- **Pourquoi c'est important** : Propose des métriques standardisées pour évaluer les harnesses agentiques — utile pour comparer Claude Code, Codex et équivalents de manière rigoureuse.
