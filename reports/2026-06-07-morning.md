## 🔥 Breaking

> Releases majeures, annonces qui changent la donne

### Microsoft Build 2026 — MAI-Thinking-1 & MAI-Code-1-Flash
- **Quoi** : Microsoft lance ses deux premiers modèles maison entraînés sans données OpenAI — MAI-Thinking-1 (raisonnement, 35B params actifs, MoE 1T params, contexte 256K) et MAI-Code-1-Flash (code, 5B params, déployé dans tous les plans GitHub Copilot)
- **Lien** : https://microsoft.ai/news/introducing-mai-thinking-1/ | https://www.neowin.net/news/microsoft-unveils-mai-thinking-1-reasoning-and-mai-code-1-coding-models/
- **Pourquoi c'est important** : MAI-Thinking-1 atteint 97 % sur AIME 2025 et rivalise avec Claude Opus 4.6 sur SWE-Bench Pro — Microsoft coupe le cordon OpenAI. MAI-Code-1-Flash bat Claude Haiku 4.5 de +16 pts sur SWE-Bench Pro en consommant 60 % moins de tokens.
- **Testable** : Oui — MAI-Code-1-Flash disponible maintenant via GitHub Copilot (tous plans) ; MAI-Thinking-1 en private preview sur Microsoft Foundry

### Anthropic dépose son S-1 pour l'IPO

### OpenAI Dreaming V3 — Mise à jour mémoire majeure
- **Quoi** : OpenAI déploie Dreaming V3, son plus grand upgrade mémoire depuis le lancement de ChatGPT, avec une page de résumé mémoire et une mise à jour automatique dans le temps ("tu vas à Singapour en juillet" → "tu es allé à Singapour")

### Windows Agent Framework open-sourcé — MIT

- **Testable** : Oui — github.com/microsoft/agent-framework
