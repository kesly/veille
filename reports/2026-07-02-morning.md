## 🔥 Breaking
> Releases majeures, annonces qui changent la donne

### Claude Fable 5 & Mythos 5 — Retour mondial après levée des contrôles d'export
- **Quoi** : Le DoC américain a levé le 30 juin les contrôles d'export imposés 18 jours plus tôt sur Claude Fable 5 et Mythos 5 ; Fable 5 est de nouveau disponible mondialement dès le 1er juillet sur Claude.ai, Claude Platform et Claude Code.
- **Lien** : https://www.anthropic.com/news/redeploying-fable-5 — https://www.cnbc.com/2026/06/30/anthropic-says-trump-admin-has-lifted-export-controls-on-claude-fable-5-and-mythos-5.html
- **Pourquoi c'est important** : Fable 5 est le modèle le plus puissant d'Anthropic, bloqué suite à un jailbreak détecté par des chercheurs d'Amazon qui permettait d'identifier des vulnérabilités logicielles. La solution finale : un unique filtre de sécurité ciblé, validé par le Center for AI Standards and Innovation du DoC. Mythos 5 reste limité aux organisations US sélectionnées. Les utilisateurs Pro/Max/Team bénéficient de Fable 5 jusqu'à 50 % de leurs limites hebdomadaires jusqu'au 7 juillet.
- **Testable** : Oui — claude.ai (disponible maintenant)

### Claude Sonnet 5 — Nouveau modèle agentic flagship
- **Quoi** : Anthropic lance Claude Sonnet 5, le Sonnet le plus agentic à ce jour, avec des améliorations majeures sur le raisonnement, l'utilisation d'outils, le coding et le knowledge work vs Sonnet 4.6.
- **Lien** : https://releasebot.io/updates/anthropic
- **Pourquoi c'est important** : Sonnet est la "colonne vertébrale" de la plupart des usages API et des agents en production — une amélioration significative impacte directement les applications agentic déployées à grande échelle.
- **Testable** : Oui — API Anthropic et Claude.ai

### GPT-5.6 (Sol / Terra / Luna) — Preview limité à 20 partenaires
- **Quoi** : OpenAI déploie GPT-5.6 en accès anticipé à 20 partenaires de confiance via API et Codex. Nouveau système de nommage : génération identifiée par le numéro (5.6), tiers de capacité par Sol/Terra/Luna.
- **Lien** : https://cryptobriefing.com/openai-releases-gpt-56-models-to-20-partners-public-launch-expected-by-july-2026/ — https://releasebot.io/updates/openai
- **Pourquoi c'est important** : GPT-5.6 promet de meilleures performances en raisonnement, coding, biologie et cybersécurité, avec un "ultra mode", des protections de sécurité renforcées et un pricing revu. Le déploiement public (ChatGPT + API large) est annoncé pour "bientôt".
- **Testable** : Non (preview partenaires uniquement — accès grand public imminent)

## 🆕 Nouveaux Outils & Produits

### Claude Science — Workbench IA pour chercheurs scientifiques
- **Quoi** : Anthropic lance Claude Science, un environnement de recherche intégré qui réunit outils courants des chercheurs, agents spécialisés pour l'analyse de littérature et l'exécution multi-étapes, accès computing flexible, et artefacts auditables.
- **Lien** : https://www.anthropic.com/news/claude-science-ai-workbench — https://techcrunch.com/2026/06/30/anthropics-claude-science-bets-on-workflow-not-a-new-model-to-win-over-scientists/
- **Pourquoi c'est important** : Ce n'est pas un nouveau modèle mais un produit vertical dédié à la science (pharma, biologie, recherche académique). Il tourne sur les modèles Claude existants (dont Opus 4.8). Programme de grants : jusqu'à 50 projets financés à hauteur de $30 000 en crédits — candidatures ouvertes jusqu'au 15 juillet 2026.
- **Testable** : Oui — anthropic.com (accès sur demande) ; grants à https://anthropic.com/science

### Google Antigravity — IDE AI-native gratuit (remplace Gemini Code Assist)
- **Quoi** : Google lance Antigravity, un IDE AI-native en public preview offrant un accès gratuit à Claude Opus 4.5, Gemini et GPT, via une interface unifiée. Remplace Gemini Code Assist pour les utilisateurs individuels (migration effective depuis le 18 juin).
- **Lien** : https://leaddev.com/ai/best-ai-coding-assistants
- **Pourquoi c'est important** : Accès multi-modèles gratuitement dans un IDE — concurrence directe à Cursor et Claude Code. Tactique agressive de Google pour récupérer les développeurs perdus.
- **Testable** : Oui — public preview gratuit

### MCP 2026-07-28 Spec — Release Candidate disponible
- **Quoi** : La plus grande révision du Model Context Protocol depuis son lancement est en RC, avec finalisation le 28 juillet. Architecture stateless (suppression des sessions), nouvelle couche d'autorisation basée sur des RFCs standards, framework d'extensions formel avec identifiants reverse-DNS.
- **Lien** : https://workos.com/blog/mcp-2026-spec-agent-authentication
- **Pourquoi c'est important** : La suppression des sessions (`Mcp-Session-Id`) simplifie massivement le déploiement en production. L'autorisation passe de "possible si tout est câblé manuellement" à "suivez ces RFCs et ça marche". Fenêtre de migration : 10 semaines. Si vous opérez des serveurs MCP en prod, c'est votre timeline.
- **Testable** : RC à lire sur le repo officiel MCP ; implémentation SDK en cours

### Databox MCP — Business data dans Claude via MCP
- **Quoi** : Databox lance un serveur MCP permettant de connecter les données business (KPIs, dashboards) directement dans Claude.
- **Lien** : https://blog.mean.ceo/ai-product-launches-news-july-2026/

- **Testable** : Oui — via Databox
