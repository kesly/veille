## 🔥 Breaking

> Releases majeures, annonces qui changent la donne

### Claude Fable 5 — Anthropic ouvre son modèle Mythos au grand public
- **Quoi** : Anthropic a lancé Claude Fable 5 le 9 juin, son premier modèle de classe Mythos accessible publiquement, avec jusqu'à 128k tokens en sortie
- **Lien** : [Anthropic - Claude Fable 5](https://www.anthropic.com/claude/fable) | [The Hacker News](https://thehackernews.com/2026/06/anthropic-releases-claude-fable-5-its.html) | [TechCrunch](https://techcrunch.com/2026/06/09/anthropic-released-claude-fable-5-its-most-powerful-model-publicly-days-after-warning-ai-is-getting-too-dangerous/)
- **Pourquoi c'est important** : Fable 5 peut gérer des tâches de plusieurs jours de façon asynchrone. Les requêtes cyber/bio sont redirigées vers Claude Opus 4.8 pour des raisons de sécurité. Disponible sur API, Claude Code, AWS, Google Cloud et Microsoft Foundry
- **Tarif** : $10/M tokens en entrée, $50/M en sortie — inclus gratuitement dans les plans Pro/Max/Team jusqu'au 22 juin, puis crédits d'usage requis
- **Testable** : Oui — via [claude.ai](https://claude.ai) sur plan Pro ou via l'API (model id: `claude-fable-5`)

- **Lien** : [OpenAI](https://openai.com/index/openai-to-acquire-ona/) | [Bloomberg](https://www.bloomberg.com/news/articles/2026-06-11/openai-to-acquire-cloud-platform-ona-to-support-ai-agents) | [CNBC](https://www.cnbc.com/2026/06/11/open-ai-ona-acquisition-codex.html)
- **Pourquoi c'est important** : Codex compte déjà 5 millions d'utilisateurs hebdomadaires (+400%). Ona apporte l'infrastructure pour que les agents IA travaillent de façon autonome pendant des heures/jours. Décision stratégique majeure pour l'ère des agents longs
- **Testable** : Non — intégration à venir dans Codex

## 🔬 Research

### TurboQuant (Google / ICLR 2026) — compression KV cache révolutionnaire
- **Quoi** : Google dévoile TurboQuant à ICLR 2026, un algorithme combinant PolarQuant et Quantized Johnson-Lindenstrauss pour réduire drastiquement la mémoire du KV cache des LLMs
- **Lien** : [Crescendo AI](https://www.crescendo.ai/news/latest-ai-news-and-updates)
- **Pourquoi c'est important** : Permet aux modèles avec de grandes fenêtres contextuelles de tourner beaucoup plus efficacement — impact direct sur les coûts d'inférence et l'IA on-device
- **Testable** : Non — papier de recherche

### O-Voxel — modélisation 3D générative de nouvelle génération
- **Quoi** : Une équipe de Tsinghua University + Microsoft Research présente O-Voxel, une nouvelle représentation pour la génération 3D capturant formes complexes et attributs de surface avec une précision inédite
- **Lien** : [Crescendo AI](https://www.crescendo.ai/news/latest-ai-news-and-updates)
- **Pourquoi c'est important** : Les assets générés surpassent les modèles existants — implications pour la création de contenu 3D, jeux vidéo, design industriel
- **Testable** : Non — papier de recherche

### Microsoft Agent Framework — MCP, CodeAct et multi-agents (Build 2026)
- **Quoi** : Microsoft annonce à Build 2026 des mises à jour de son Agent Framework open-source (fusion AutoGen + Semantic Kernel), avec intégrations MCP, CodeAct, et workflows multi-agents en .NET et Python
- **Lien** : [Microsoft DevBlogs](https://devblogs.microsoft.com/agent-framework/microsoft-agent-framework-at-build-2026-announce/)
- **Pourquoi c'est important** : SDK unifié pour construire des agents en entreprise, interopérable avec les standards MCP — répond directement à l'écosystème LangGraph / CrewAI
- **Testable** : Oui — open-source
