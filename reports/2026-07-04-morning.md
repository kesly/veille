## 🧠 Mises à jour de Modèles

### Gemini 3.1 Flash Image + Gemini 3 Pro Image
- **Quoi** : Google a sorti le 30 juin deux nouveaux modèles de génération d'images : Gemini 3.1 Flash Image ($0.50/$3.00 par M tokens) et Gemini 3 Pro Image ($2.00/$12.00 par M tokens)
- **Lien** : https://llm-stats.com/llm-updates
- **Pourquoi c'est important** : Génération d'images nativement intégrée à l'écosystème Gemini API, compétition directe avec DALL-E et Midjourney via API
- **Testable** : Oui — Google AI Studio, Gemini API

### Grok 4.5 en beta privée
- **Quoi** : Elon Musk a confirmé le 28 juin que Grok 4.5 tourne en beta privée (SpaceX, Tesla), sur une architecture V9 avec ~1,5 trillion de paramètres
- **Lien** : https://releasebot.io/updates/xai
- **Pourquoi c'est important** : Montée en gamme massive par rapport à Grok 4.3 (flagship public actuel) ; lancement public probablement dans les semaines à venir
- **Testable** : Non (beta privée)

### Codex CLI v0.130
- **Quoi** : Mise à jour du Codex CLI ajoutant l'authentification AWS Bedrock et ne nécessitant plus de redémarrage de l'éditeur
- **Lien** : https://releasebot.io/updates/openai
- **Pourquoi c'est important** : Intégration Bedrock simplifiée pour les équipes entreprise utilisant AWS
- **Testable** : Oui — `npm install -g @openai/codex@latest`

### MCP Spec 2026 Release Candidate : passage au stateless
- **Quoi** : Le Release Candidate de la spécification MCP 2026-07-28 supprime les sessions au niveau protocole (plus de `Mcp-Session-Id`), passant à un core HTTP stateless
- **Lien** : https://byteiota.com/mcp-goes-stateless-2026-release-candidate/
- **Pourquoi c'est important** : Les serveurs MCP peuvent désormais tourner derrière un load balancer round-robin classique sans session sticky ni shared store — scalabilité dramatiquement simplifiée
- **Testable** : Oui — RC disponible, spec finale prévue le 28 juillet
