### MCP Release Candidate 2026-07-28 — Rupture majeure du protocole
- **Quoi** : Le Model Context Protocol annonce son Release Candidate pour le 28 juillet 2026 — la plus grande révision depuis le lancement. Breaking changes multiples : cœur stateless, nouveau système d'Extensions, Tasks, MCP Apps, et refonte OAuth/OpenID Connect
- **Lien** : https://blog.modelcontextprotocol.io/posts/2026-07-28-release-candidate/ — https://tokenmix.ai/blog/mcp-updates-changelog-every-protocol-change-2026
- **Pourquoi c'est important** : Tout MCP Server existant devra migrer avant fin juillet. Les SDK Tier 1 ont 10 semaines pour shipper le support. Le protocole stateless change fondamentalement l'architecture : fini les connexions longues, place aux requêtes HTTP ordinaires scalables. Le nouveau système d'Extensions permet des UIs server-rendues dans les clients MCP
- **Testable** : Oui — RC disponible maintenant sur modelcontextprotocol.io pour tests de migration
