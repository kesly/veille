# AI Watch — 2026-07-18 Matin

## 🔥 Breaking
> Releases majeures, annonces qui changent la donne

### Kimi K3 — Moonshot AI (sorti le 16 juillet)
- **Quoi** : Le plus grand modèle open-weight jamais publié — 2,8 billions de paramètres MoE (896 experts, 16 activés/token), contexte 1M tokens, multimodal texte + images
- **Lien** : https://venturebeat.com/technology/chinas-moonshot-ai-releases-kimi-k3-the-largest-open-source-model-ever-rivaling-top-u-s-systems
- **Pourquoi c'est important** : Prend la 1ère place du Frontend Code Arena (1 679 pts), devançant Claude Fable 5 (1 631), GPT-5.6 Sol (1 618). Performances comparables à Opus 4.8. Poids ouverts promis pour le 27 juillet.
- **Testable** : Oui — API disponible à 3 $/M tokens entrée, 15 $/M tokens sortie. https://kimik3.xyz/

### Claude Sonnet 5 — Anthropic (nouveau défaut Free/Pro)
- **Quoi** : Remplace Opus 4.8 comme modèle par défaut pour les utilisateurs Free et Pro
- **Lien** : https://llm-stats.com/llm-updates
- **Pourquoi c'est important** : 63,2 % sur SWE-bench Pro, bat Opus 4.8 sur Terminal-Bench 2.1 (80,4 % vs 74,6 %), à seulement 2 $/10 $ par million de tokens jusqu'au 31 août. Meilleur rapport qualité/prix du marché.
- **Testable** : Oui — claude.ai et API Anthropic (prix promotionnel actif)

### MCP Spec Release Candidate — Publication le 28 juillet
- **Quoi** : Plus grande révision du protocole MCP depuis son lancement : core stateless sur HTTP, extensions MCP Apps (UI server-rendered) et Tasks (tâches longues)
- **Lien** : https://blog.modelcontextprotocol.io/posts/2026-07-28-release-candidate/
- **Pourquoi c'est important** : Standardise l'écosystème de 5 800+ serveurs communautaires, facilite le déploiement à l'échelle sans infrastructure spéciale
- **Testable** : Non — RC disponible pour commentaires, final le 28 juillet

## 🆕 Nouveaux Outils & Produits

### GPT-Live — OpenAI (lancé le 8 juillet)
- **Quoi** : Nouvelle génération de modèles vocaux full-duplex (écoute et parle simultanément) pour ChatGPT
- **Lien** : https://openai.com/index/introducing-gpt-live/
- **Pourquoi c'est important** : Architecture full-duplex = conversations naturelles sans latence de tour. Comparable à un vrai échange téléphonique.
- **Testable** : Oui — disponible dans l'app ChatGPT

### Grok Voice Agent Builder — xAI (lancé le 1er juillet)
- **Quoi** : Plateforme no-code pour créer des agents vocaux de production sur Grok Voice (speech-to-speech). Description en langage naturel + outils = agent téléphonique en ~2 minutes
- **Lien** : https://www.digitalapplied.com/blog/grok-voice-agent-builder-no-code-voice-agents-2026
- **Pourquoi c'est important** : Tarif unique $0,05/min tout inclus, barrière d'entrée quasi nulle pour les voice agents en prod
- **Testable** : Oui — bêta ouverte sur la plateforme xAI

### AgentPrizm — AgentMemory + AgentSkills (lancé le 9 juillet)
- **Quoi** : Plateforme REST API + MCP pour donner aux agents une mémoire persistante entre sessions et un catalogue de compétences réutilisables
- **Lien** : https://agentic.ai/news
- **Pourquoi c'est important** : Comble le manque de mémoire cross-session des agents LLM sans avoir à construire soi-même l'infrastructure
- **Testable** : Oui — API disponible

### Safari MCP Server — Apple (lancé le 1er juillet)
- **Quoi** : Premier serveur MCP officiel d'un grand éditeur de navigateur — 16 outils donnant aux agents IA accès direct à Web Inspector de Safari
- **Lien** : https://developer.chrome.com/blog/chrome-devtools-mcp
- **Pourquoi c'est important** : Les agents de dev peuvent maintenant déboguer, inspecter et interagir avec Safari nativement via MCP
- **Testable** : Oui — disponible dans Safari 20+

### Meta Muse Image & Muse Video (lancé le 7 juillet)
- **Quoi** : Générateur d'images IA de Meta Superintelligence Labs (gratuit dans Meta AI, Instagram Stories, WhatsApp) + modèle vidéo en développement
- **Lien** : https://about.fb.com/news/2026/07/introducing-muse-image-meta-ai/
- **Pourquoi c'est important** : Distribution directe à des milliards d'utilisateurs TikTok/Instagram. Polémique : génération de visages de personnes avec comptes Instagram publics sans consentement explicite.
- **Testable** : Oui — gratuit via Meta AI app

### J-Space — Anthropic
- **Quoi** : Découverte d'un espace interne aux LLM ("J-space") contenant des mots absents de l'output mais influençant le raisonnement du modèle

- **Pourquoi c'est important** : Ouvre une nouvelle voie d'interprétabilité mécanistique pour comprendre "ce que le modèle pense" avant d'écrire
- **Testable** : Non — recherche publiée, pas encore d'outil public

### Google DeepMind — Raisonnement mathématique niveau IMO
- **Quoi** : Le dernier système de raisonnement mathématique de DeepMind score dans le top 1 % sur les problèmes des Olympiades Internationales de Mathématiques
- **Lien** : https://skycrumbs.com/blog/ai-research-july-2026
- **Pourquoi c'est important** : Dépasse le niveau humain expert sur des problèmes de concours parmi les plus difficiles au monde
- **Testable** : Non — résultats de recherche

- **Quoi** : Nouvelle méthode d'entraînement qui apprend aux modèles à n'activer que les paramètres les plus pertinents pour chaque tâche

- **Testable** : Non — papier de recherche
