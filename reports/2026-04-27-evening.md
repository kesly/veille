# AI Watch — 2026-04-27 Soir

## 🔥 Breaking

> Releases majeures et annonces qui changent la donne

### GPT-5.5 — disponible dans l'API dès aujourd'hui
- **Quoi** : OpenAI a mis GPT-5.5 et GPT-5.5 Pro en accès API général le 24 avril ; premier modèle entièrement re-entraîné depuis GPT-4.5.
- **Lien** : https://openai.com/index/introducing-gpt-5-5/
- **Pourquoi c'est important** : Terminal-Bench 2.0 à 82,7 % (vs Claude Opus 4.7 à 69,4 %), Expert-SWE à 73,1 % sur des tâches de 20 h de travail humain. Intégré nativement dans Codex et ChatGPT.
- **Testable** : Oui — API disponible, plans Plus/Pro/Business/Enterprise dans ChatGPT.

### DeepSeek V4 Preview — open-source MIT, 1,6 T paramètres
- **Quoi** : DeepSeek a lancé V4-Flash (284 B / 13 B actifs) et V4-Pro (1,6 T / 49 B actifs), MoE, contexte 1 M tokens, open-weights MIT.
- **Lien** : https://api-docs.deepseek.com/news/news260424
- **Pourquoi c'est important** : Premier modèle DeepSeek optimisé pour les puces Huawei Ascend ; performances égales aux modèles fermés sur math/code. Signal fort dans la guerre sino-américaine des puces IA.
- **Testable** : Oui — API DeepSeek et poids disponibles sur Hugging Face.

### Stanford AI Index 2026 — publié aujourd'hui
- **Quoi** : Stanford HAI a publié ce matin son rapport annuel sur l'état de l'IA.
- **Lien** : https://hai.stanford.edu/ai-index/2026-ai-index-report
- **Pourquoi c'est important** : SWE-bench Verified est passé de 60 % à quasi 100 % en un an. 88 % des organisations utilisent l'IA. Les meilleurs modèles dépassent les humains sur des questions de niveau doctorat en sciences. Transparence en chute libre (score moyen à 40/100 contre 58 l'an dernier).
- **Testable** : Non — rapport PDF téléchargeable.

### Microsoft / OpenAI : deal révisé — OpenAI libéré du cloud Azure
- **Quoi** : Microsoft et OpenAI ont amendé leur accord ; OpenAI peut désormais déployer ses services sur n'importe quel cloud provider, Microsoft cesse de percevoir une part des revenus.
- **Lien** : https://llm-stats.com/ai-news
- **Pourquoi c'est important** : Rupture majeure dans l'exclusivité Azure. OpenAI peut viser Google Cloud, AWS, etc. Redistribution des cartes dans l'infrastructure IA.
- **Testable** : Non.
