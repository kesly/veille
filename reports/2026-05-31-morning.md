### Inworld AI — Realtime TTS-2 (Research Preview)
- **Quoi** : **Realtime TTS-2** est un modèle vocal en boucle fermée qui prend en entrée l'audio réel des tours précédents (pas un transcript) — il "entend" vraiment l'interlocuteur.
- **Lien** : https://inworld.ai/blog/realtime-tts-2

- **Testable** : Oui — via Inworld API (research preview)

### Augment Code — Intent (macOS)
- **Quoi** : **Intent** est un workspace multi-agent autonome pour macOS où un agent Coordinateur décompose les tâches en spec vivante et délègue à des agents spécialistes qui travaillent en parallèle dans des workspaces isolés.
- **Lien** : https://www.augmentcode.com/tools/8-top-ai-coding-assistants-and-their-best-use-cases
- **Pourquoi c'est important** : Première solution production-ready de multi-agent agentic coding sur desktop — différente de Cursor ou Copilot qui restent en mono-agent.
- **Testable** : Oui — disponible sur macOS

### xAI Grok Build 0.1 — Coding API (beta publique)
