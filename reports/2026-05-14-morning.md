## 🆕 Nouveaux Outils & Produits

### AWS MCP Server — Disponibilité Générale
- **Quoi** : Le serveur MCP d'AWS passe en GA le 6 mai : les agents de code accèdent à l'ensemble des APIs AWS via un seul outil MCP, avec contrôles IAM, logs CloudTrail et métriques CloudWatch
- **Lien** : https://aws.amazon.com/about-aws/whats-new/2026/05/aws-mcp-server/
- **Pourquoi c'est important** : Toute l'infrastructure cloud AWS devient manipulable directement par des agents IA en production, sans wrapper custom — standard MCP Linux Foundation désormais à 9 400+ serveurs dans le registre
- **Testable** : Oui — gratuit (facturation des ressources AWS utilisées)

### Inworld Realtime TTS-2 — Voix adaptative contextuelle
- **Quoi** : Nouveau modèle TTS pour agents conversationnels en temps réel : capte le ton, le rythme et l'état émotionnel de l'utilisateur, se dirige avec du langage naturel ("fatigué mais chaleureux après une longue journée"), latence < 200ms, 100+ langues, une seule identité vocale
- **Lien** : https://inworld.ai/blog/realtime-tts-2
- **Pourquoi c'est important** : TTS 1.5 d'Inworld est déjà #1 sur l'Artificial Analysis Speech Arena (devant Google #2 et ElevenLabs #3) — TTS-2 pousse encore plus loin avec adaptation émotionnelle temps-réel; intégrations natives Vapi, Pipecat, LiveKit
- **Testable** : Oui — research preview via Inworld API

### Sierra AI — $950M levés pour les agents enterprise
- **Quoi** : Sierra lève $950M mené par Tiger Global et GV, valorisation post-money >$15B
- **Lien** : https://techcrunch.com/2026/05/04/sierra-raises-950m-as-the-race-to-own-enterprise-ai-gets-serious/
- **Pourquoi c'est important** : Confirme que les agents vocaux/texte pour le service client enterprise sont devenus un marché de plusieurs milliards — Sierra est la référence B2B face à des plateformes plus génériques
- **Testable** : Non — enterprise only

### CopilotKit — $27M Series A pour agents in-app
- **Quoi** : CopilotKit lève $27M (Glilot Capital, NFX, SignalFire) pour permettre aux développeurs de déployer des agents IA natifs directement dans leurs applications SaaS
- **Lien** : https://techcrunch.com/2026/05/05/copilotkit-raises-27m-to-help-devs-deploy-app-native-ai-agents/
- **Pourquoi c'est important** : Open-source, SDK React — alternative concrète pour intégrer un co-pilote IA à n'importe quelle app web sans construire l'infra soi-même
- **Testable** : Oui — open-source, docs sur copilotkit.ai

### Hugging Face ml-intern — Agent ML open-source autonome
- **Quoi** : Hugging Face publie ml-intern, un agent capable de lire des papers, entraîner des modèles et livrer du code ML en autonomie via l'écosystème HF (datasets, compute cloud, docs)
- **Lien** : https://github.com/huggingface/ml-intern
- **Pourquoi c'est important** : Premier agent ML "chercheur → code → déploiement" entièrement open-source signé par le leader de l'écosystème open IA
- **Testable** : Oui — open-source, gratuit

## 🧠 Mises à jour de Modèles

### GPT-5.5 Instant — Nouveau modèle par défaut de ChatGPT
- **Quoi** : Remplace GPT-5.3 Instant comme modèle gratuit par défaut depuis le 5 mai ; 52,5% moins d'hallucinations sur des prompts à enjeux élevés (médecine, droit, finance), 30,2% moins verbeux, personnalisation renforcée via historique, fichiers et Gmail connecté
- **Lien** : https://openai.com/index/gpt-5-5-instant/
- **Pourquoi c'est important** : Amélioration notable de la fiabilité factuelle sans coût supplémentaire — GPT-5.3 Instant reste accessible aux payants encore 3 mois
- **Testable** : Oui — ChatGPT gratuit (chat.openai.com)

### OpenAI GPT-Realtime-Whisper & GPT-Realtime-Translate
- **Quoi** : Deux nouveaux modèles temps-réel : STT streaming (GPT-Realtime-Whisper) et traduction live speech-to-speech (GPT-Realtime-Translate, 70+ langues en entrée → 13 langues de sortie)
- **Lien** : https://releasebot.io/updates/openai
- **Pourquoi c'est important** : Ouvre la porte aux applications multilingues en temps réel via l'API OpenAI, sans pipeline externe de traduction
- **Testable** : Oui — via OpenAI API (Realtime API)

## 🔬 Research

### AEC-Bench — Benchmark agents IA pour l'Architecture et la Construction
- **Quoi** : Nouveau benchmark pour évaluer les agents IA sur des tâches réelles dans les domaines Architecture, Ingénierie et Construction (AEC)
- **Lien** : https://richlyai.com/blog/aec-bench-ai-benchmark-for-architecture-construction-ai-news/
- **Pourquoi c'est important** : Premier benchmark vertical sérieux pour un secteur à fort potentiel de disruption par les agents autonomes ; permet de calibrer les choix de modèles sur des tâches métier concrètes
- **Testable** : Non — dataset en cours de publication

### MolmoAct2 — Robotique open : action reasoning multimodal
- **Quoi** : Modèle d'action open-source pour la robotique : backbone vision-langage, tokenizer d'actions open-weight, raisonnement adaptatif pour réduire la latence de décision
- **Lien** : https://arxiv.org/list/cs.AI/current
- **Pourquoi c'est important** : Rare modèle robotique entièrement open, concurrent des approches propriétaires RT-2 et Pi0
- **Testable** : Oui — open-weight sur HuggingFace

### Contamination des benchmarks — problème systémique confirmé
- **Quoi** : Les benchmarks classiques (MMLU, etc.) sont de plus en plus contaminés par les données d'entraînement ; émergence de "Humanity's Last Exam" (2 500 questions académiques multimodales) comme référence plus robuste
- **Lien** : https://hai.stanford.edu/ai-index/2026-ai-index-report/technical-performance
- **Pourquoi c'est important** : Rend difficile la comparaison objective des modèles — à garder en tête lors de l'interprétation de tout classement benchmark
