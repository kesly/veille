## 🆕 Nouveaux Outils & Produits

### Voicebox — Studio de synthèse vocale open-source (aujourd'hui)
- **Quoi** : Voicebox est un studio local de clonage vocal open-source, alternative gratuite à ElevenLabs. 5 moteurs TTS (dont Qwen3-TTS, LuxTTS, Chatterbox), 23 langues, éditeur timeline multi-voix.
- **Lien** : https://github.com/jamiepine/voicebox — https://voicebox.sh/
- **Pourquoi c'est important** : Tourne entièrement en local (Tauri/Rust, pas Electron), zéro envoi de données, API REST intégrée. Support macOS MLX, Windows CUDA, Linux, AMD ROCm. Lancé aujourd'hui — analyse du jour sur AIToolly.
- **Testable** : Oui — open-source, gratuit, installable via GitHub releases.

### Claw Code — Coding agent open-source, 72K+ stars en quelques jours
- **Quoi** : Framework de coding agent open-source, réécriture Python/Rust de l'architecture Claude Code, signé par Sigrid Jin (profil WSJ, 25 milliards de tokens traités).
- **Lien** : https://github.com/ultraworkers/claw-code — https://claw-code.codes/
- **Pourquoi c'est important** : Parmi les repos ayant grandi le plus vite dans l'histoire de GitHub (72K stars, 72K forks en quelques jours). Supporte tous les providers LLM. Attention : supply-chain attack signalée sur des clones npm le 31 mars — utiliser uniquement le repo officiel.
- **Testable** : Oui — open-source, gratuit (pip install depuis le repo officiel).

### Telnyx LiveKit on Telnyx — Infrastructure voice AI sous-200ms
- **Quoi** : Telnyx lance une plateforme hébergée intégrant LiveKit sur sa propre infra GPU + téléphonie, pour des agents vocaux à latence < 200ms round-trip.
- **Lien** : https://telecomreseller.com/2026/04/10/telnyx-launches-livekit-on-telnyx-for-deploying-voice-ai-agents-with-lower-cost-and-ultra-low-latency/
- **Pourquoi c'est important** : 50% moins cher que LiveKit Cloud, audio jamais hors du réseau Telnyx, STT/TTS colocalisés avec les points de présence téléphoniques mondiaux. Idéal pour call centers / agents vocaux produits.
- **Testable** : Oui — offre payante avec essai, docs sur telnyx.com.

### Visa Intelligent Commerce Connect — Paiements pour agents IA
