# AI Watch — 2026-04-29 Soir

## 🔥 Breaking

### GPT-5.5 — OpenAI
- **Quoi** : Nouveau modèle OpenAI optimisé pour le coding, les tâches ordinateur et la recherche avancée, 1M tokens de contexte
- **Lien** : https://openai.com/index/introducing-gpt-5-5/
- **Pourquoi c'est important** : Arrive moins de 2 mois après GPT-5.4. Disponible en API dès le 24 avril, déployé aux abonnés Plus/Pro/Business/Enterprise. GPT-5.4 Thinking a déjà franchi le seuil humain sur OSWorld (75%) et GDPVal (83% sur 44 métiers qualifiés). La cadence de release s'accélère dramatiquement.
- **Testable** : Oui — ChatGPT (abonnés payants) et API OpenAI

### DeepSeek V4 Preview — DeepSeek (open-source)
- **Quoi** : Modèle flagship open-source en deux variantes : V4-Pro (1.6T/49B params actifs) et V4-Flash (284B/13B actifs), 1M tokens de contexte, licence MIT
- **Lien** : https://api-docs.deepseek.com/news/news260424
- **Pourquoi c'est important** : Bat tous les modèles open actuels en Math/STEM/Coding, rivale les meilleurs modèles closed-source. Prix ultra-compétitif : $0.14/M tokens (Flash) à $1.74/M (Pro). Architecture inédite (Compressed Sparse Attention, Muon optimizer). MIT = usage commercial libre.
- **Testable** : Oui — API DeepSeek disponible dès maintenant

### Meta Muse Spark — Pivot stratégique majeur
- **Quoi** : Meta abandonne la philosophie open-weight et lance son premier modèle propriétaire, Muse Spark, sous la direction d'Alexandr Wang (nouveau Chief AI Officer)
- **Lien** : https://www.cnbc.com/2026/04/08/meta-debuts-first-major-ai-model-since-14-billion-deal-to-bring-in-alexandr-wang.html
- **Pourquoi c'est important** : Renversement stratégique historique — Meta quitte l'open-source pour concurrencer directement OpenAI et Google. Meta annonce aussi $115–135 milliards de capex IA en 2026, quasi-doublement vs. 2025.
- **Testable** : Accès limité pour l'instant

### Google investit jusqu'à $40B dans Anthropic
- **Quoi** : $10B immédiats + $30B conditionnels à des jalons de performance, valorisant Anthropic à $350B
- **Lien** : https://www.cnbc.com/2026/04/24/google-to-invest-up-to-40-billion-in-anthropic-as-search-giant-spreads-its-ai-bets.html
- **Pourquoi c'est important** : Anthropic dépasse $30B ARR (contre ~$9B fin 2025). Google sécurise son accès aux modèles frontier face à OpenAI. Accord Amazon séparé de $25B également actif.
- **Testable** : Non

- **Quoi** : Modèle TTS capable de générer 60s d'audio en 1 seconde, voix personnalisée à partir de quelques secondes d'audio de référence
