# AI Watch — 2026-04-30 Matin

## 🔥 Breaking
> Releases majeures, annonces qui changent la donne

### OpenAI GPT-5.5 — Le modèle omnimodal natif
- **Quoi** : OpenAI lance GPT-5.5, son modèle le plus intelligent à ce jour, nativement omnimodal (texte, images, audio, vidéo dans un seul système unifié)
- **Lien** : [openai.com/index/introducing-gpt-5-5](https://openai.com/index/introducing-gpt-5-5/)
- **Pourquoi c'est important** : Contexte de 1M tokens, latence identique à GPT-5.4 mais performances nettement supérieures, et plus efficace en tokens sur les tâches Codex. Disponible en API le 24 avril 2026.
- **Prix API** : $5 / 1M tokens input — $30 / 1M tokens output (batch à -50%)
- **Testable** : Oui — API disponible, ChatGPT Plus/Pro, Amazon Bedrock (preview)

### DeepSeek V4 — L'open-source qui rivalise avec les frontières
- **Quoi** : DeepSeek lance deux modèles open-source sous licence MIT : V4-Pro (1,6T paramètres) et V4-Flash (284B), avec fenêtre de contexte native de 1M tokens
- **Lien** : [api-docs.deepseek.com/news/news260424](https://api-docs.deepseek.com/news/news260424)
- **Pourquoi c'est important** : V4-Pro score 80,6% sur SWE-bench Verified (vs 80,8% pour Claude Opus 4.6), et 3 206 sur Codeforces (record mondial pour un modèle). Architecture hybride CSA+HCA qui réduit les FLOPs à seulement 27% de V3.2 en contexte 1M tokens. Poids complets sur Hugging Face, compatible API OpenAI et Anthropic.
- **Testable** : Oui — API DeepSeek, poids sur [huggingface.co/deepseek-ai/DeepSeek-V4-Pro](https://huggingface.co/deepseek-ai/DeepSeek-V4-Pro)

### OpenAI débarque sur AWS Bedrock — Fin de l'exclusivité Microsoft
- **Quoi** : Le 28 avril, après la fin de l'accord d'exclusivité Microsoft-OpenAI (7 ans), GPT-5.5 et GPT-5.4 arrivent sur Amazon Bedrock en preview
- **Lien** : [cnbc.com — OpenAI brings models to AWS](https://www.cnbc.com/2026/04/28/openai-brings-models-to-aws-after-ending-exclusivity-with-microsoft.html)
- **Pourquoi c'est important** : Rupture structurelle du marché cloud IA. Microsoft garde des droits non-exclusifs sur l'IP d'OpenAI jusqu'en 2032. OpenAI peut désormais vendre sur n'importe quel cloud. Impact massif sur la concurrence Azure/AWS/GCP.
- **Testable** : Oui — Amazon Bedrock (preview), [aboutamazon.com](https://www.aboutamazon.com/news/aws/bedrock-openai-models)

### Google investit $40 milliards dans Anthropic
- **Quoi** : Google injecte $10 milliards immédiatement dans Anthropic (valorisation $350B) avec jusqu'à $30 milliards supplémentaires conditionnés à des objectifs de performance, plus 5 gigawatts de compute sur 5 ans
- **Lien** : [techcrunch.com — Google to invest up to $40B in Anthropic](https://techcrunch.com/2026/04/24/google-to-invest-up-to-40b-in-anthropic-in-cash-and-compute/)
- **Pourquoi c'est important** : Repositionne radicalement le rapport de force dans la course IA. Google (qui développe Gemini en concurrent direct) devient le principal bailleur d'Anthropic. Signal que l'écosystème se consolide autour de 2-3 acteurs majeurs.
- **Testable** : Non — investissement stratégique

## 🧠 Mises à jour de Modèles

### Claude Mythos Preview — Le modèle trop dangereux pour être public
- **Quoi** : Anthropic annonce Claude Mythos Preview (7 avril), modèle général de pointe avec capacités cybersécurité exceptionnelles — accès restreint, non public
- **Lien** : [red.anthropic.com/2026/mythos-preview](https://red.anthropic.com/2026/mythos-preview/)
- **Pourquoi c'est important** : Mythos a exploité de façon autonome une CVE FreeBSD vieille de 17 ans (RCE avec root via NFS). +31 points vs Opus 4.6 sur l'Olympiade Mathématique US 2026. Anthropic a lancé le **Projet Glasswing** pour utiliser Mythos à sécuriser les logiciels critiques mondiaux plutôt que de le rendre public.
- **Testable** : Non — accès restreint, projet Glasswing uniquement

## 🔬 Research

### AutoResearchBench — Évaluer les agents de recherche scientifique
- **Quoi** : Nouveau benchmark évaluant la capacité des agents IA à mener des recherches scientifiques autonomes (deep research + wide research sur papers)
- **Lien** : [devflokers.com — AI papers April 2026](https://www.devflokers.com/blog/new-ai-papers-arxiv-last-24-hours-april-2026)
- **Pourquoi c'est important** : Les LLMs les plus puissants obtiennent encore des scores faibles sur ce benchmark, suggérant que la recherche scientifique autonome reste non résolue. Référence pour évaluer les "deep research" agents.
- **Testable** : Oui — benchmark public

### CodeSOTA Speech Leaderboard 2026
- **Quoi** : Mise à jour du leaderboard STT/TTS (18 modèles chacun) — STT atteint la précision humaine sur audio propre, TTS passe le test en aveugle de naturalité
- **Lien** : [codesota.com/speech](https://www.codesota.com/speech)
- **Pourquoi c'est important** : Jalons symboliques : la voix IA devient indiscernable de la voix humaine selon des critères objectifs. Référence pour choisir le bon modèle STT/TTS.
- **Testable** : Oui — leaderboard en ligne

### DeepSeek V4 — Architecture CSA+HCA
- **Quoi** : Le paper DeepSeek V4 décrit une architecture d'attention hybride (Compressed Sparse Attention + Heavily Compressed Attention) permettant de traiter 1M tokens avec seulement 27% des FLOPs de V3.2
- **Lien** : [datacamp.com/blog/deepseek-v4](https://www.datacamp.com/blog/deepseek-v4)
- **Pourquoi c'est important** : Innovation architecturale significative pour le contexte long. Poids ouverts = reproductible par la communauté.
- **Testable** : Oui — poids sur Hugging Face
