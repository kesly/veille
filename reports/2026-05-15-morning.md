## 🧠 Mises à jour de Modèles

### Gemini Omni — Modèle multimodal unifié repéré en bêta
- **Quoi** : Gemini Omni est apparu dans des tests internes de l'app Gemini (12 mai). Il unifierait génération texte + image + vidéo dans une seule pipeline. Les samples montrent : édition vidéo in-chat, object swap, suppression de watermarks, forte adhérence aux prompts et mouvements fluides — au niveau de Runway Gen-3

- **Testable** : Non encore — annonce attendue le 19 mai

### Grok 4.3 — Nouveau standard xAI API
- **Quoi** : Grok 4.3 est depuis mai le modèle phare de xAI, optimisé pour l'agentic tool calling. Pricing : $1.25/1M input, $2.50/1M output. Remplace 8 modèles antérieurs
- **Lien** : https://aifreeforever.com/blog/grok-4-3-is-now-live-on-the-xai-api-it-is-best-for-agentic-tool-calling

### GPT-Realtime-2 / Translate / Whisper — Suite voix OpenAI dans l'API

## 🔬 Research

### "Attention Is All You Need V2" — Google reforge l'architecture transformer
- **Quoi** : Des chercheurs Google ont publié une nouvelle architecture adressant les faiblesses des transformeurs classiques : catastrophic forgetting (oubli catastrophique lors du fine-tuning) et memory degradation sur les longues séquences
- **Lien** : https://llm-stats.com/ai-news
- **Pourquoi c'est important** : Si l'architecture tient ses promesses, ce serait la première révision fondamentale du transformer depuis 2017 — impact potentiel sur tous les futurs modèles
- **Testable** : Non — recherche fondamentale, pas encore de modèle publié

### SubQ 1M-Preview — Premier LLM commercial subquadratique
- **Quoi** : Subquadratic AI lance le premier modèle commercial à architecture d'attention sparse subquadratique, avec une fenêtre de contexte native de 12 millions de tokens. Revendique ~1/5 du coût des modèles frontier et jusqu'à 52x plus rapide sur l'attention à grande échelle
- **Lien** : https://whatllm.org/blog/new-ai-models-may-2026
- **Pourquoi c'est important** : La barrière du contexte long et coûteux pourrait tomber — applications RAG, analyse de codebase entière, analyse légale de corpus volumineux deviennent économiquement viables
- **Testable** : Oui — disponible en preview sur la plateforme Subquadratic AI

### MCP Protocol — Donné à la Linux Foundation
- **Quoi** : Le protocole MCP d'Anthropic, qui a franchi 97 millions d'installations en mars 2026, a été cédé à la Linux Foundation pour une gouvernance neutre. Tous les grands providers IA livrent désormais des outils MCP-compatibles
- **Lien** : https://www.crescendo.ai/news/latest-ai-news-and-updates
- **Pourquoi c'est important** : MCP devient une vraie norme ouverte (comme HTTP ou TCP/IP) — les trois nouveaux serveurs MCP du jour (Mobbin, Pacvue, iManage) illustrent cette dynamique
- **Testable** : Non — gouvernance organisationnelle

### OpenHuman — IA personnelle locale & privée sur GitHub
- **Quoi** : OpenHuman (tinyhumansai) émerge sur GitHub Trending : projet de "superintelligence personnelle" locale, basé sur les principes privacy-first, simplicity et haute performance. Conçu pour tourner sans cloud
- **Lien** : https://aitoolly.com/ai-news/article/2026-05-13-openhuman-a-new-private-and-powerful-personal-ai-superintelligence-project-emerges-on-github
- **Pourquoi c'est important** : Tendance croissante des solutions IA on-device et privacy-first comme alternative aux LLMs cloud — intéressant pour les usages sensibles
- **Testable** : Oui — open source sur GitHub
