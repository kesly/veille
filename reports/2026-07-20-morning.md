# AI Watch — 2026-07-20 Matin

## 🔥 Breaking
> Releases majeures, annonces qui changent la donne

### MCP Specification Release Candidate — Refonte majeure du protocole
- **Quoi** : La spec MCP 2026-07-28 RC est disponible — plus grande révision depuis le lancement, avec un cœur stateless sur HTTP standard, extensions UI (MCP Apps), Tasks, et OAuth/OIDC natif
- **Lien** : https://blog.modelcontextprotocol.io/posts/2026-07-28-release-candidate/
- **Pourquoi c'est important** : Changement architectural profond — les serveurs MCP passent au HTTP stateless, ouvre la voie à du déploiement serverless et scalable sans state persistant. Politiques de déprécation formelles = maturité industrielle
- **Testable** : Oui — RC disponible, migration docs en ligne

### Grok Build — CLI coding agent open-source par xAI
- **Quoi** : xAI lance Grok Build, agent de coding CLI open-source (Apache 2.0) basé sur Grok 4.5, avec terminal UI, gestion de contexte transparente, plugins, skills et MCP
- **Lien** : https://github.com/trending
- **Pourquoi c'est important** : Premier coding agent CLI de xAI, code entièrement open-source — concurrent direct de Claude Code et Aider, avec accès à Grok 4.5 (meilleure perf coding à ce jour selon xAI)
- **Testable** : Oui — open-source, installable via npm/cargo

### Anthropic lève les rate limits — réponse à la guerre des prix
- **Quoi** : Anthropic supprime les caps d'usage (5h et hebdo) sur Claude Free/Pro pour retenir les développeurs face aux nouveaux entrants low-cost
- **Lien** : https://www.anthropic.com/news/redeploying-fable-5
- **Pourquoi c'est important** : Signal fort : la guerre des prix initiée début juillet (GPT-5.6 Luna à $6/M tokens out, Grok 4.5) force Anthropic à jouer la rétention plutôt que la monétisation agressive
- **Testable** : Oui — immédiatement sur claude.ai
