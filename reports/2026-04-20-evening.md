## 🔥 Breaking

> Releases majeures, annonces qui changent la donne

### Claude Mythos — Trop dangereux pour être publié
- **Quoi** : Anthropic annonce Claude Mythos (10 000 milliards de paramètres), mais décide de NE PAS le publier après avoir découvert sa capacité à identifier et enchaîner des dizaines de milliers de vulnérabilités zero-day.
- **Lien** : [Bloomberg — How Anthropic Discovered Mythos Was Too Dangerous](https://www.bloomberg.com/news/features/2026-04-16/how-anthropic-discovered-mythos-ai-was-too-dangerous-for-release)
- **Pourquoi c'est important** : Scores records : 93,9 % sur SWE-bench Verified, 94,6 % sur GPQA Diamond. Disponible uniquement via **Project Glasswing** (partenaires Amazon, Microsoft, Apple, Google, Nvidia) avec 100 M$ de crédits pour la cyberdéfense. Premier modèle frontier officiellement jugé trop risqué pour une sortie publique.
- **Testable** : Non — accès restreint aux partenaires Glasswing uniquement.

### xAI lance les APIs Grok STT & TTS (19 avril 2026)
- **Quoi** : xAI publie des APIs voix standalone (Speech-to-Text + Text-to-Speech) ciblant les développeurs enterprise.
- **Lien** : [MarkTechPost — xAI Launches Standalone Grok STT & TTS APIs](https://www.marktechpost.com/2026/04/18/xai-launches-standalone-grok-speech-to-text-and-text-to-speech-apis-targeting-enterprise-voice-developers/)
- **Pourquoi c'est important** : STT en 25 langues avec taux d'erreur de 5 % (vs ElevenLabs 12 %, Deepgram 13,5 %). TTS 20 langues, 5 voix expressives avec balises émotionnelles (`[laugh]`, `<whisper>`), tarifé à **$4,20/million de caractères**. xAI entre frontalement dans le marché ElevenLabs/Deepgram/AssemblyAI.
- **Testable** : Oui — via l'API xAI (docs.x.ai).

### Microsoft Agent Framework 1.0 GA (3 avril 2026)
- **Quoi** : Microsoft unifie Semantic Kernel et AutoGen en un seul SDK .NET + Python avec support natif MCP et A2A.
- **Lien** : [Digital Applied — Microsoft Agent Framework 1.0](https://www.digitalapplied.com/blog/microsoft-agent-framework-1-0-dotnet-python-guide)
- **Pourquoi c'est important** : 75 000+ étoiles GitHub combinées sur les projets précédents, APIs stables long-terme, interopérabilité native avec l'écosystème d'agents 2026 (MCP + A2A). Devient le standard Microsoft pour l'orchestration d'agents.
- **Testable** : Oui — open source, SDK disponible sur NuGet et PyPI.
