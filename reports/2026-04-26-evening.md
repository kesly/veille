- **Quoi** : Alphabet injecte immédiatement 10 Md$ dans Anthropic avec 30 Md$ supplémentaires conditionnels aux jalons de développement, valorisant la société entre 350 et 380 Md$.

### DeepSeek V4 Preview — open source, 1.6T paramètres, contexte 1M tokens
- **Quoi** : DeepSeek lance deux modèles MoE en preview : V4-Pro (1.6T params, 49B activés) et V4-Flash (284B params, 13B activés), contexte natif 1 million de tokens, open source sur Hugging Face.
- **Lien** : https://api-docs.deepseek.com/news/news260424
- **Pourquoi c'est important** : V4-Pro atteint 80.6% sur SWE-bench Verified (à 0.2 points de Claude Opus 4.6) à seulement 3,48 $/M tokens de sortie contre 25 $ pour Claude — rapport qualité/prix historique. Premier modèle open source bâti nativement autour du 1M-token context.
- **Testable** : Oui — API disponible sur api.deepseek.com, poids sur Hugging Face (deepseek-ai/DeepSeek-V4-Pro)

### OpenAI ferme la Sora app aujourd'hui (26 avril)
- **Quoi** : La version grand public de Sora (interface sociale + UI principale) est mise hors ligne aujourd'hui ; l'API sera fermée le 24 septembre 2026.
- **Lien** : https://resource.digen.ai/sora-2-openai-shutdown-guide-2026/
- **Pourquoi c'est important** : Signale qu'OpenAI abandonne le marché consumer de la vidéo pour recentrer sur les outils enterprise et son "super app" GPT-5.5. Opportunité pour Runway, Kling, Veo 3.1 de récupérer la base utilisateurs.
- **Testable** : Non — la fermeture est effective
