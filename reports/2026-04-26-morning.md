## 🔥 Breaking

### GPT-5.5 — OpenAI lance son nouveau flagship

- **Quoi** : OpenAI a lancé GPT-5.5 le 23 avril, disponible pour tous les abonnés payants (Plus, Pro, Business, Enterprise) et via API depuis le 24 avril
- **Lien** : https://openai.com/index/introducing-gpt-5-5/ — https://www.cnbc.com/2026/04/23/openai-announces-latest-artificial-intelligence-model.html
- **Pourquoi c'est important** : Nettement amélioré sur le coding, l'utilisation d'ordinateurs (computer use) et la recherche approfondie. GPT-5.5 Pro est accessible en API. C'est le modèle qui propulse Codex.
- **Testable** : Oui — via ChatGPT (abonnés) et l'API OpenAI (`gpt-5.5`, `gpt-5.5-pro`)

### DeepSeek V4 — Preview lancé, 1T paramètres, open-source

- **Quoi** : DeepSeek a publié le 24 avril un preview de V4, en deux variantes : V4 Flash (284B total / 13B actifs) et V4 Pro (1,6T total / 49B actifs), contexte 1M tokens, 384K output max
- **Lien** : https://api-docs.deepseek.com/news/news260424 — https://www.cnbc.com/2026/04/24/deepseek-v4-llm-preview-open-source-ai-competition-china.html
- **Pourquoi c'est important** : Technique "Hybrid Attention Architecture" pour les longues conversations. Open-source. Tarifs agressifs : Flash à $0,14/M tokens input, Pro à $1,74/M. Rivalise avec les meilleurs modèles occidentaux à fraction du prix.
- **Testable** : Oui — API DeepSeek (`deepseek-v4-flash`, `deepseek-v4-pro`), téléchargement local possible

### Anthropic Managed Agents — Plateforme agent hébergée en bêta publique

- **Quoi** : Anthropic lance Managed Agents, un service de plateforme Claude pour les tâches agents long-horizon avec sandboxing, gestion d'état, exécution d'outils et récupération d'erreurs natifs
- **Lien** : https://www.anthropic.com/news
- **Pourquoi c'est important** : Première offre cloud d'Anthropic pour déployer des agents sans gérer l'infrastructure. Sessions durables, outil d'accès sécurisé, démarrage rapide. Concurrent direct d'OpenAI Agents SDK.
- **Testable** : Oui — via la plateforme développeur Anthropic (bêta publique)

## 🆕 Nouveaux Outils & Produits

### Hugging Face ml-intern — Agent ML open-source qui bat Claude Code

- **Quoi** : Hugging Face lance ml-intern (21 avril), un agent ML engineer open-source qui automatise le post-training LLM — lecture de papers arXiv, recherche de datasets, entraînement, évaluation
- **Lien** : https://github.com/huggingface/ml-intern — https://www.marktechpost.com/2026/04/21/hugging-face-releases-ml-intern-an-open-source-ai-agent-that-automates-the-llm-post-training-workflow/
- **Pourquoi c'est important** : En démo, il a poussé Qwen3-1.7B de 10% à 32% sur GPQA en moins de 10h. Surpasse Claude Code sur le raisonnement scientifique. $1 000 en crédits GPU et Anthropic offerts aux premiers utilisateurs.
- **Testable** : Oui — CLI + web app disponibles, open-source (gratuit)

### xAI Grok STT & TTS APIs — Voice AI concurrent d'ElevenLabs

- **Quoi** : xAI a lancé le 18 avril deux APIs audio standalone : Grok STT (Speech-to-Text) et Grok TTS (Text-to-Speech), 20+ langues, 5 voix expressives, REST/WebSocket
- **Lien** : https://blog.laozhang.ai/en/posts/grok-speech-to-text-api — https://docs.x.ai/developers/release-notes
- **Pourquoi c'est important** : Tarif $0,10/h (batch) et $0,20/h (streaming STT). Intégration native avec Grok reasoning + tool-calling + recherche web en temps réel. Même infra que Grok Voice sur mobile/Tesla/Starlink.
- **Testable** : Oui — `POST https://api.x.ai/v1/stt` via console.x.ai (payant, tarif accessible)

### Google Deep Research Max — Recherche autonome avec support MCP

- **Pourquoi c'est important** : Deep Research Max utilise du calcul étendu à l'inférence pour itérer sur des recherches complexes. Support MCP = les agents peuvent connecter leurs propres sources de données.
- **Testable** : Oui — via l'API Gemini (payant)

- **Quoi** : Cognition a intégré Devin dans Windsurf IDE (15 avril) : délégation de tâches depuis l'éditeur local vers l'agent cloud. Nouveaux plans tarifaires (Free, Pro, Max, Teams, Enterprise).

### OpenAI Agents SDK — Mise à jour majeure (15 avril)

- **Quoi** : OpenAI a livré une mise à jour substantielle de l'Agents SDK : sandboxing isolé, harness long-horizon, orchestration de sous-agents, support provider-agnostique (100+ LLMs)

### Google Colab MCP Server — Agents IA dans les notebooks

- **Quoi** : Google lance un MCP server officiel pour Colab : les agents IA peuvent créer des notebooks, exécuter du code, gérer des dépendances et renvoyer des résultats
