### Claude Fable 5 — Anthropic ouvre son modèle Mythos au grand public
- **Quoi** : Anthropic a mis Claude Fable 5 en disponibilité générale le 9 juin — un modèle de classe Mythos rendu safe pour le grand public, avec des safeguards qui basculent automatiquement sur Opus 4.8 pour les requêtes sensibles (<5 % des sessions).
- **Lien** : https://www.anthropic.com/news/claude-fable-5-mythos-5
- **Pourquoi c'est important** : C'est la première fois qu'un modèle de niveau Mythos est accessible sans accréditation spéciale. Benchmarks état-de-l'art sur quasi tous les domaines (code, vision, science, long-context). Prix : $10/M tokens input, $50/M tokens output — soit moitié moins que Claude Mythos Preview. Accès gratuit sur les plans Pro/Max/Team jusqu'au 22 juin, puis usage credits.
- **Testable** : Oui — déjà disponible sur claude.ai et l'API Anthropic. Model ID : `claude-fable-5`.

### Microsoft Build 2026 — 7 modèles MAI maison, indépendants d'OpenAI
- **Quoi** : Microsoft a dévoilé une famille de 7 modèles développés en interne (MAI) lors de Build 2026 (2-3 juin) : MAI-Thinking-1 (raisonnement, 35B params, 256K ctx), MAI-Code-1-Flash (5B params, 51% SWE-Bench Pro), MAI-Voice-2 (TTS, 15 langues), MAI-Image-2.5 (text-to-image), MAI-Transcribe-1.5 (STT, 43 langues).
- **Lien** : https://microsoft.ai/news/building-a-hillclimbing-machine-launching-seven-new-mai-models/
- **Pourquoi c'est important** : Pivot stratégique majeur vers l'autonomie face à OpenAI. Tarifs 20-60 % inférieurs aux modèles OpenAI équivalents sur Azure. MAI-Code-1-Flash est intégré nativement dans GitHub Copilot et VS Code.
- **Testable** : Oui — via Azure AI Foundry et GitHub Copilot.
