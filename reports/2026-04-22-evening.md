# AI Watch — 2026-04-22 Soir

## 🔥 Breaking

### Claude Mythos — le modèle le plus puissant d'Anthropic (accès restreint)
- **Quoi** : Anthropic a confirmé l'existence de Mythos, son modèle le plus capable à ce jour, entraîné sur 10 000 milliards de paramètres, avec 93,9 % sur SWE-bench Verified et 94,6 % sur GPQA Diamond.
- **Lien** : https://www.usnews.com/news/top-news/articles/2026-04-20/explainer-what-do-we-know-about-anthropics-mythos-amid-rising-concerns
- **Pourquoi c'est important** : Mythos est capable d'identifier et d'exploiter de façon autonome des dizaines de milliers de vulnérabilités zero-day dans les OS et navigateurs majeurs. Anthropic limite volontairement son accès à un petit cercle d'organisations après consultation avec des gouvernements.
- **Testable** : Non — accès fermé, recherche preview uniquement pour partenaires sélectionnés.

### Amazon injecte jusqu'à 25 milliards de dollars dans Anthropic
- **Quoi** : Amazon finalise un accord d'investissement de 5 Md$ immédiats + jusqu'à 20 Md$ supplémentaires liés à des jalons commerciaux, en échange d'un engagement d'Anthropic à dépenser 100 Md$ sur AWS sur 10 ans (5 GW de capacité de calcul).
- **Lien** : https://techcrunch.com/2026/04/20/anthropic-takes-5b-from-amazon-and-pledges-100b-in-cloud-spending-in-return/
- **Pourquoi c'est important** : Le deal repositionne AWS comme infrastructure principale d'entraînement et d'inférence pour Claude, avec une dépendance croisée massive entre les deux géants. Anthropic engagerait aussi une utilisation des TPUs Broadcom/Google à partir de 2027.
- **Testable** : Non — deal infrastructurel.

### ChatGPT Images 2.0 (GPT-Image-2) — l'IA qui "pense" avant de dessiner
- **Quoi** : OpenAI a lancé le 21 avril ChatGPT Images 2.0 avec le modèle gpt-image-2 : raisonnement natif, résolution 2K, batches de 8 images cohérentes, support multilingue et texte enfin rendu correctement.
- **Lien** : https://openai.com/index/introducing-chatgpt-images-2-0/
- **Pourquoi c'est important** : Premier modèle d'image avec du "thinking" intégré à l'architecture. Classé #1 sur Image Arena avec 242 points d'avance. API disponible dès le lancement. La sortie a fait chuter l'action Figma de 7,3 % le jour J.
- **Testable** : Oui — disponible dans ChatGPT (tous utilisateurs pour le mode standard, Plus/Pro/Business pour le mode thinking). API disponible.

- **Quoi** : Microsoft a sorti la version stable 1.0 de son framework agent multi-plateforme (.NET + Python) avec APIs stables, LTS, support MCP natif, et une DevUI browser pour visualiser l'exécution en temps réel.

### xAI Grok STT & TTS APIs — voix enterprise
- **Quoi** : xAI a lancé des APIs standalone de Speech-to-Text (25 langues, batch et streaming) et Text-to-Speech (5 voix expressives, 20 langues, tags de style [laugh], [sigh], <whisper>), basées sur l'infrastructure Grok Voice.
- **Lien** : https://www.marktechpost.com/2026/04/18/xai-launches-standalone-grok-speech-to-text-and-text-to-speech-apis-targeting-enterprise-voice-developers/
- **Pourquoi c'est important** : Le STT Grok affiche 5 % de taux d'erreur sur les appels téléphoniques, soit 2,4× moins que ElevenLabs (12 %). TTS à 4,20 $ / million de caractères. Concurrent direct d'ElevenLabs et Deepgram.
- **Testable** : Oui — disponible via xAI API (docs.x.ai).

### Google Veo 3.1 Lite — génération vidéo la plus accessible

## 🔬 Research

### Anthropic — 171 patterns "d'émotions fonctionnelles" dans Claude (2 avril)
- **Quoi** : L'équipe d'interprétabilité d'Anthropic a identifié 171 patterns d'activation dans Claude Sonnet 4.5 correspondant à des états fonctionnels similaires aux émotions, qui influencent causalement les sorties du modèle.
- **Lien** : https://www.anthropic.com/news
- **Pourquoi c'est important** : Première étude à échelle industrielle montrant que des "états internes" non-subjectifs mais causaux existent dans les LLMs. Ouvre des questions sur l'alignement, la sécurité et l'éthique de l'IA.
- **Testable** : Non — paper de recherche.

### Google TurboQuant à ICLR 2026 — compression modèles ×100 moins cher
- **Quoi** : Google présente TurboQuant, algorithme de quantification vectorielle qui réduit massivement l'overhead mémoire des modèles, permettant de faire tourner des LLMs frontier sur matériel réduit.
- **Lien** : https://www.sciencedaily.com/releases/2026/04/260405003952.htm
- **Pourquoi c'est important** : Si les chiffres se confirment en production, cela démocratise radicalement le déploiement de grands modèles on-premise et edge.
- **Testable** : Non — publication académique, implémentation en cours.

### AI + Quantum — chiffrement internet menacé plus tôt que prévu
- **Quoi** : Des recherches de Google et de la startup Oratomic suggèrent que des ordinateurs quantiques capables de casser les protocoles de chiffrement internet pourraient arriver bien avant les estimations précédentes, l'IA ayant été "instrumentale" dans le développement de l'algorithme.
- **Lien** : https://time.com/article/2026/04/07/ai-quantum-computing-advance/
- **Pourquoi c'est important** : Accélère l'urgence de la migration vers la cryptographie post-quantique (PQC). Signal fort pour les équipes sécurité.
- **Testable** : Non — recherche académique.
