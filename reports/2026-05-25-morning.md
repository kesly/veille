- **Testable** : Oui — Gemini app et API

### Gemini Omni (Google)
- **Quoi** : Nouveau modèle multimodal "tout-en-tout" : n'importe quelle entrée (image, audio, vidéo, texte) vers n'importe quelle sortie en commençant par la vidéo ; ancré dans la connaissance du monde réel

- **Pourquoi c'est important** : Premier modèle grand public à génération vidéo native intégrée à un LLM multimodal — rivalise directement avec Sora/Veo
- **Testable** : Oui — Gemini app (disponible maintenant)

### GPT-5.5 Instant (OpenAI) — Nouveau défaut ChatGPT
- **Quoi** : Modèle léger et faible latence, nouveau défaut dans ChatGPT depuis le 5 mai 2026 ; 52,5% moins d'hallucinations que GPT-5.3 Instant sur médecine, droit, finance ; personnalisation améliorée
- **Lien** : https://openai.com/index/gpt-5-5-instant/
- **Pourquoi c'est important** : Tous les utilisateurs gratuits bénéficient maintenant d'un modèle plus fiable par défaut — la barre d'hallucinations descend pour l'usage grand public
- **Testable** : Oui — ChatGPT (gratuit)

### SubQ 1M-Preview (Subquadratic)
- **Quoi** : Premier LLM commercial à architecture sparse-attention entièrement sous-quadratique ; contexte natif de 12 millions de tokens ; ~1/5 du coût des modèles frontier ; attention jusqu'à 52x plus rapide à l'échelle

- **Pourquoi c'est important** : Rupture architecturale potentielle — si les benchmarks tiennent, ce modèle remet en question l'hypothèse que les transformers standard sont la seule voie
- **Testable** : Oui — preview commercial disponible

### OpenAI Voice API — Raisonnement + Traduction
- **Quoi** : OpenAI publie dans son API de nouveaux modèles vocaux temps-réel capables de raisonner, traduire et transcrire — au-delà du simple TTS
