## 🆕 Nouveaux Outils & Produits

### Vercel Eve — framework agent open-source (17 juin)
- **Quoi** : Framework TypeScript pour construire des agents IA ; chaque agent = un dossier de fichiers qui compile vers Vercel Functions
- **Lien** : https://vercel.com/blog/introducing-eve
- **Pourquoi c'est important** : Exécution durable, sandbox, human-in-the-loop, evals — tout inclus par défaut ; Apache 2.0 ; s'autoproclame "Next.js pour les agents"
- **Testable** : Oui — GitHub open-source, beta publique disponible

### Grok Imagine 1.5 + Video 1.5 — xAI (17 juin)
- **Quoi** : Nouvelle version de Grok Imagine avec saut de qualité visuelle ; grok-imagine-video-1.5 désormais GA sur l'API xAI
