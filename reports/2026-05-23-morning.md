## 🔥 Breaking
> Releases majeures, annonces qui changent la donne

### Andrej Karpathy rejoint Anthropic
- **Quoi** : Co-fondateur d'OpenAI et ex-directeur IA de Tesla rejoint Anthropic pour mener une nouvelle équipe de pretraining qui utilise Claude pour accélérer la recherche sur l'entraînement des modèles de base
- **Lien** : https://www.technologyreview.com/2026/05/21/1137735/anthropics-code-with-claude-showed-off-codings-future-whether-you-like-it-or-not/
- **Pourquoi c'est important** : Coup de tonnerre — Karpathy est l'une des figures les plus influentes de l'IA mondiale. Son départ d'OpenAI (dont il est co-fondateur) et son arrivée chez le concurrent direct est le transfert de talent le plus significatif de l'année. Signale qu'Anthropic intensifie la course aux modèles fondamentaux.
- **Testable** : Non

### Google I/O 2026 — Gemini 3.5 Flash + Gemini Omni + WebMCP
- **Quoi** : Google lance Gemini 3.5 Flash (4x plus rapide que les modèles frontier, surpasse Gemini 3.1 Pro sur presque tous les benchmarks), Gemini Omni (génération vidéo native depuis tout type d'input), et propose WebMCP — un standard web ouvert permettant aux pages d'exposer des outils MCP directement dans Chrome
- **Lien** : https://developers.googleblog.com/all-the-news-from-the-google-io-2026-developer-keynote/
- **Pourquoi c'est important** : Gemini 3.5 Flash redéfinit le rapport vitesse/intelligence pour les agents en production. Gemini Omni ouvre la génération vidéo native. WebMCP (origin trial Chrome 149) est potentiellement transformateur : le web entier devient adressable par des agents IA.
- **Testable** : Oui — Gemini 3.5 Flash via Gemini API et AI Studio ; WebMCP via Chrome 149 origin trial

### Anthropic — MCP Tunnels & Self-hosted Sandboxes
- **Quoi** : Anthropic déploie deux nouvelles fonctionnalités pour Claude Managed Agents : les self-hosted sandboxes (public beta) permettant l'exécution des outils sur l'infra du client, et les MCP Tunnels (research preview) donnant aux agents accès aux systèmes internes privés sans les exposer sur internet
- **Lien** : https://claude.com/blog/claude-managed-agents-updates
- **Pourquoi c'est important** : Résout le principal blocage à l'adoption enterprise des agents IA — sécurité et conformité. Les organisations peuvent connecter leurs bases de données internes, APIs privées et systèmes de ticketing à Claude sans ouvrir leur réseau. Le tunnel chiffre end-to-end via une seule connexion sortante, sans règle de pare-feu entrante.
- **Testable** : Oui (sandboxes en public beta via Cloudflare, Daytona, Modal, Vercel) / Sur demande (MCP Tunnels)

## 🧠 Mises à jour de Modèles

### Gemini 3.5 Flash (Google)
- **Quoi** : Surpasse Gemini 3.1 Pro sur presque tous les benchmarks tout en étant 4x plus rapide ; optimisé pour les workflows agentiques temps-réel
- **Lien** : https://cloud.google.com/blog/products/ai-machine-learning/innovations-from-google-io-26-on-google-cloud
- **Pourquoi c'est important** : Nouveau standard de référence vitesse/intelligence pour les APIs de production — Google récupère l'avantage sur ce segment
- **Testable** : Oui — Gemini API / AI Studio

### Gemini Omni (Google)
- **Quoi** : Nouveau modèle multimodal de Google capable de générer n'importe quel type de sortie à partir de n'importe quel input (texte, image, audio, vidéo) — déployé dans Gemini App, YouTube Shorts et Flow
- **Lien** : https://techcrunch.com/2026/05/19/googles-gemini-omni-turns-images-audio-and-text-into-video-and-thats-just-the-start/
- **Pourquoi c'est important** : Génération vidéo native intégrée dans un modèle de conversation — 10 secondes de vidéo par requête, accessible via les produits grand public de Google
- **Testable** : Oui — Gemini App / YouTube Shorts

### GPT-5.5 Instant (OpenAI)
- **Quoi** : Modèle par défaut de ChatGPT depuis le 5 mai — réponses plus précises, hallucinations réduites, meilleure personnalisation
- **Lien** : https://techcrunch.com/2026/05/05/openai-releases-gpt-5-5-instant-a-new-default-model-for-chatgpt/
- **Pourquoi c'est important** : Tous les utilisateurs ChatGPT (y compris gratuits) bénéficient maintenant d'un modèle nettement amélioré par rapport à GPT-5
- **Testable** : Oui — ChatGPT (tous plans)

### Grok 4.3 (xAI)
- **Quoi** : Flagship économique de xAI — raisonnement intégré, fenêtre de contexte 1M tokens, input vidéo natif. Déployé le 4 mai.
- **Lien** : https://medium.com/nlplanet/xai-releases-grok-4-3-weekly-ai-newsletter-may-4th-2026-4b7e8fea0f10
- **Pourquoi c'est important** : 1M tokens + vidéo native dans un modèle cost-efficient = nouvelle baseline pour l'analyse de documents longs et le traitement multimodal à moindre coût
- **Testable** : Oui — xAI API / SuperGrok

### Voxtral TTS (Mistral AI) — Open Weights
- **Quoi** : Modèle TTS open-weights 4B paramètres, 9 langues, adapte une voix à partir de 3s de référence seulement. Bat ElevenLabs en évaluation humaine : Hindi 79.8%, Espagnol 87.8%, Arabe 72.9%. Disponible sur HuggingFace (CC BY-NC 4.0).
- **Lien** : https://mistral.ai/news/voxtral-tts
- **Pourquoi c'est important** : Premier TTS open-source qui bat officiellement ElevenLabs sur des évaluations humaines. Tourne sur un seul GPU 16GB via vLLM — déployable on-premise. API à $0.016/1k caractères.
- **Testable** : Oui — mistralai/Voxtral-4B-TTS-2603 sur HuggingFace + API Mistral

### Inworld Realtime TTS-2
- **Quoi** : Modèle vocal à architecture closed-loop — prend l'audio réel des tours précédents en entrée (pas la transcription), adapte dynamiquement le ton et les nuances. Classé #1 sur l'Artificial Analysis Speech Arena.
- **Lien** : https://www.marktechpost.com/2026/05/05/inworld-ai-launches-realtime-tts-2-a-closed-loop-voice-model-that-adapts-to-how-you-actually-talk/
- **Pourquoi c'est important** : Avance architecturale significative : cohérence vocale cross-tours sans retranscription, pour des agents voix plus naturels
- **Testable** : Oui — Inworld API (research preview)
