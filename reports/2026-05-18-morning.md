## 🔥 Breaking

### Google I/O 2026 — Keynote demain (19-20 mai)
- **Quoi** : Google tient son I/O annuel demain — annonces majeures attendues sur Gemini, Android et les outils IA
- **Lien** : https://io.google/2026/
- **Pourquoi c'est important** : Gemini Omni (fusion texte + image + vidéo dans une seule interface), potentiel Gemini 3.5 ou 4.0, Veo upgrades, Gemma 4, Genie 3 (robotique), Gemini Intelligence sur Android. C'est l'événement IA de la semaine — possibilité de repositionnement face à GPT-5.5.
- **Testable** : Oui — livestream public sur io.google. Certaines fonctionnalités probablement disponibles immédiatement après l'annonce.

### Anthropic "Dreaming" — Les agents Claude apprennent entre les sessions
- **Quoi** : Anthropic lance "Dreaming", un système de mémorisation curative qui permet aux agents IA de revoir leurs sessions passées, d'en extraire des patterns et de s'améliorer automatiquement
- **Lien** : https://venturebeat.com/technology/anthropic-introduces-dreaming-a-system-that-lets-ai-agents-learn-from-their-own-mistakes
- **Pourquoi c'est important** : Harvey (legal AI) a vu ses taux de complétion de tâches augmenter de **~6x** ; Wisedocs (medical) a réduit son temps de review de **50%**. C'est un pas concret vers des agents qui s'auto-améliorent en production.
- **Testable** : Oui — disponible en research preview sur la plateforme Claude. Outcomes et multi-agent orchestration en public beta.

### SubQ 1M-Preview — Premier LLM subquadratique commercial (12M tokens)
- **Quoi** : Subquadratic lance SubQ, le premier LLM commercial basé sur une architecture sparse-attention subquadratique avec une fenêtre de contexte native de 12 millions de tokens
- **Lien** : https://siliconangle.com/2026/05/05/subquadratic-launches-29m-bring-12m-token-context-windows-ai/
- **Pourquoi c'est important** : ~1 000x moins de compute que les modèles frontier classiques à contexte équivalent. Sur RULER 128K : 95% d'accuracy pour **8 $** vs **2 600 $** pour Claude Opus. Architecture fondamentalement différente du Transformer classique. Seed de $29M.
- **Testable** : Oui — preview disponible via Subquadratic

## 🆕 Nouveaux Outils & Produits

### OpenAI Daybreak — Plateforme cybersécurité AI-powered
- **Quoi** : OpenAI lance Daybreak, une plateforme de détection de vulnérabilités et de validation de patches basée sur Codex Security + GPT-5.5
- **Lien** : https://thehackernews.com/2026/05/openai-launches-daybreak-for-ai-powered.html
- **Pourquoi c'est important** : Intègre threat modeling, secure code review, patch validation et dependency analysis dans le workflow dev quotidien. Partenaires : Akamai, Cisco, CrowdStrike, Palo Alto, Cloudflare, Fortinet. Lancé le 11 mai — le même jour où Google confirmait le **premier zero-day construit par IA**.
- **Testable** : Partiellement — accès sur demande via openai.com/daybreak

### Inworld Realtime TTS-2 — Voice AI contextuel et empathique
- **Quoi** : Inworld AI lance Realtime TTS-2, un stack vocal complet (STT → Router → TTS) sur une seule connexion WebSocket persistante
- **Lien** : https://www.marktechpost.com/2026/05/05/inworld-ai-launches-realtime-tts-2-a-closed-loop-voice-model-that-adapts-to-how-you-actually-talk/
- **Pourquoi c'est important** : Latence médiane < 200ms. Le modèle analyse le ton, le rythme et l'état émotionnel de l'utilisateur en temps réel. Route dynamiquement parmi 200+ modèles. Stack voice-agent complet prêt pour la production.
- **Testable** : Oui — API disponible via Inworld AI

### ChatGPT Finance — Connexion aux comptes bancaires (US)
- **Quoi** : OpenAI lance des outils de finances personnelles pour les abonnés ChatGPT Pro aux États-Unis via Plaid (12 000+ institutions)
- **Lien** : https://techcrunch.com/2026/05/15/openai-launches-chatgpt-for-personal-finance-will-let-you-connect-bank-accounts/
- **Pourquoi c'est important** : Fidelity, Chase, Schwab, Robinhood, Amex, Capital One — ChatGPT devient un assistant financier personnel avec accès aux données réelles. Première incursion d'OpenAI dans la fintech.
- **Testable** : Oui (US uniquement, Pro) — via ChatGPT interface web/mobile

### LangGraph v1.2 — Framework agents mis à jour
- **Quoi** : LangGraph v1.2 ajoute timeouts par nœud, error recovery gracieux, DeltaChannel pour réduire l'overhead des checkpoints, et une API de streaming v3 centrée sur les content-blocks
- **Lien** : https://www.morphllm.com/ai-agent-framework
- **Pourquoi c'est important** : Améliore significativement la fiabilité des agents long-running en production. Le DeltaChannel réduit la taille des checkpoints sur les threads longs.
- **Testable** : Oui — open source, `pip install langgraph==1.2`

### OpenClaw v2026.5.12 — Agent personnel open source trending
- **Quoi** : OpenClaw, agent personnel open source qui tourne en local et se connecte aux outils existants, vient de sortir la v2026.5.12 avec Claude Opus 4.7, Kimi K2.6, support `/think xhigh` et isolation de worker Telegram
- **Lien** : https://www.nocobase.com/en/blog/best-open-source-ai-projects-github-2026
- **Pourquoi c'est important** : Projet en tête du trending GitHub sur le topic AI. Alternative locale et self-hostable aux assistants cloud.
- **Testable** : Oui — open source, GitHub

## 🔬 Research

### Architecture SSA (Subquadratic Sparse Attention) — Rupture architecturale
- **Quoi** : SubQ de Subquadratic implémente une architecture sparse-attention propriétaire qui passe de O(n²) à O(n·log n) — la première à atteindre le niveau frontier commercialement
- **Lien** : https://explainx.ai/blog/subq-ssa-sparse-attention-12m-context-2026
- **Pourquoi c'est important** : 52x plus efficace que FlashAttention à 1M tokens. Ouvre la voie vers des contextes de 50M tokens (objectif Q4 2026). Si les claims sont vérifiés indépendamment, c'est une rupture de paradigme pour les LLM sur des documents très longs (code entier, corpus scientifiques, etc.).
- **Testable** : Oui — SubQ 1M-Preview accessible

### Anthropic "Dreaming" — Mémorisation curative pour agents
- **Quoi** : Système de mémoire inter-sessions pour agents : review automatique, déduplication, suppression des entrées obsolètes, mise en avant des patterns récurrents
- **Lien** : https://letsdatascience.com/blog/anthropic-dreaming-claude-managed-agents-self-improving-may-6
- **Pourquoi c'est important** : Premiers résultats en production mesurables (+6x Harvey, -50% Wisedocs). Ouvre la question de la convergence vers des comportements non prévus à force d'auto-amélioration.
- **Testable** : Oui — research preview Claude platform
