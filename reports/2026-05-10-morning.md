# AI Watch — 2026-05-10 Matin

## 🔥 Breaking
> Releases majeures, annonces qui changent la donne

### Claude Mythos Preview + Project Glasswing (Anthropic)
- **Quoi** : Anthropic dévoile son modèle frontier non-public *Claude Mythos Preview*, capable d'identifier des zero-days dans les principaux OS et navigateurs, et lance *Project Glasswing* pour le mettre entre les mains des défenseurs en priorité.
- **Lien** : https://www.anthropic.com/project/glasswing
- **Pourquoi c'est important** : Mythos a passé un test de cyberattaque bout-en-bout en 32 étapes — GPT-5.5 l'a suivi trois semaines après. L'UK AISI estime que la capacité offensive des modèles double tous les quatre mois. Anthropic ne le rend pas public et l'offre d'abord à un cercle restreint (AWS, Apple, Cisco, Google, JPMorgan, Microsoft, Linux Foundation, NVIDIA…) pour patcher les systèmes critiques avant une diffusion plus large.
- **Testable** : Non — accès restreint aux partenaires Project Glasswing

### GPT-5.5 Instant — nouveau modèle par défaut de ChatGPT (OpenAI, 5 mai)
- **Quoi** : OpenAI remplace GPT-5.3 Instant par GPT-5.5 Instant comme modèle par défaut pour tous les utilisateurs ChatGPT.
- **Lien** : https://openai.com/index/gpt-5-5-instant/
- **Pourquoi c'est important** : 52,5 % d'hallucinations en moins sur des prompts à enjeux élevés (médecine, droit, finance). Réponses plus concises et mieux personnalisées. GPT-5.5-Cyber, sa variante sécurité, a aussi passé le test cyber-offensif en 32 étapes.
- **Testable** : Oui — disponible directement sur ChatGPT (gratuit et payant)

### OpenAI Voice Intelligence — trois nouveaux modèles audio (7 mai)
- **Quoi** : OpenAI lance GPT-Realtime-2 (raisonnement GPT-5), GPT-Realtime-Translate (traduction live 70+ langues → 13 langues) et GPT-Realtime-Whisper (STT en streaming).
- **Lien** : https://releasebot.io/updates/openai
- **Pourquoi c'est important** : La traduction temps réel multilingue directement dans l'API ouvre des cas d'usage massifs (support client, téléconsultation, conf-calls internationaux) sans pipeline externe.
- **Testable** : Oui — via l'API OpenAI (payant)

### Anthropic dépasse OpenAI en ARR — 80x de croissance en Q1
- **Quoi** : L'ARR annualisé d'Anthropic atteint 30 milliards de dollars, devant OpenAI (24 Md$). La croissance Q1 a été 80x supérieure aux prévisions internes (qui tablaient sur 10x).
- **Lien** : https://fortune.com/2026/05/08/anthropic-80fold-growth-quarter-renting-elon-musk-data-center/
- **Pourquoi c'est important** : Signal structurel majeur : Claude gagne la bataille de l'adoption enterprise. Anthropic loue désormais la ferme Colossus d'Elon Musk (220 000 GPU Nvidia), signe un deal compute de 1,8 Md$ avec Akamai, et des engagements multi-gigawatts avec Amazon, Google/Broadcom et Microsoft/NVIDIA.
- **Testable** : Non — indicateur business

## 🆕 Nouveaux Outils & Produits

### AWS MCP Server — disponibilité générale (6 mai)
- **Quoi** : AWS lance en GA le AWS Agent Toolkit : un bundle de serveurs MCP officiels permettant aux agents IA (Claude Code, Cursor, Codex, Kiro, Windsurf…) d'appeler n'importe quelle API AWS avec gouvernance IAM, logs CloudTrail et métriques CloudWatch.
- **Lien** : https://aws.amazon.com/about-aws/whats-new/2026/05/aws-mcp-server/
- **Pourquoi c'est important** : Les agents de code peuvent maintenant provisionner et opérer de l'infrastructure AWS sans sortir du contexte de développement, avec une traçabilité enterprise native.
- **Testable** : Oui — open-source, intégrable dans tout client MCP compatible

### Mistral Workflows — orchestration IA durable (public preview)
- **Quoi** : Mistral lance *Workflows* en preview publique : orchestration Python durable avec human-in-the-loop, exécution traçable, et déploiement cloud/on-prem/hybride directement depuis Studio et Le Chat.
- **Lien** : https://mistral.ai/news
- **Pourquoi c'est important** : Première brique "production-grade" de Mistral pour passer des POC aux pipelines IA réels sans dépendre d'un orchestrateur tiers (LangGraph, Temporal…).
- **Testable** : Oui — via Mistral Studio (compte gratuit, usage payant selon modèle)

### Voxtral TTS — premier modèle TTS de Mistral (9 mai)
- **Quoi** : Mistral sort son premier modèle text-to-speech, *Voxtral TTS*, 4B paramètres, multilingue (EN, FR, ES, PT, IT, NL, DE, HI, AR), disponible via API Mistral ($0,016/1 000 chars) et en open-weight (CC BY-NC 4.0) sur Hugging Face.
- **Lien** : https://www.progressiverobot.com/2026/05/09/voxtral-tts/
- **Pourquoi c'est important** : Mistral dispose maintenant d'un stack complet speech (STT Voxtral + TTS), le tout en open-weight ce qui est rare à ce niveau de qualité.
- **Testable** : Oui — Hugging Face (gratuit, CC BY-NC) ou API Mistral (payant)

### Grok STT + TTS APIs (xAI, 6 mai)
- **Quoi** : xAI ouvre deux APIs audio standalone : *Grok Speech-to-Text* (25+ langues, même stack que Grok Voice / Tesla / Starlink) et *Grok Text-to-Speech*.
- **Lien** : https://x.ai/news/grok-stt-and-tts-apis
- **Pourquoi c'est important** : Alternative sérieuse à OpenAI Whisper et ElevenLabs, directement intégrable via l'API Grok existante.
- **Testable** : Oui — via API xAI (payant)

### Inworld Realtime TTS-2 (5 mai)
- **Quoi** : Inworld AI lance *Realtime TTS-2*, un modèle vocal qui perçoit le ton, le rythme et l'état émotionnel de l'interlocuteur et s'adapte en temps réel. STT inclus avec profilage speaker (âge, accent, pitch, émotion).
- **Lien** : https://inworld.ai/blog/realtime-tts-2
- **Pourquoi c'est important** : Premier modèle TTS "closed-loop" : il écoute la conversation entière, pas seulement le texte à synthétiser. Idéal pour les NPC de jeux vidéo et les agents vocaux conversationnels.
- **Testable** : Oui — API Inworld (plan gratuit disponible)

### Minions — tableau de bord pour agents IA en parallèle (Product Hunt #4, 8 mai)
- **Quoi** : Outil d'automatisation de workflows multi-agents avec un tableau unique pour piloter des tâches IA exécutées en parallèle.
- **Lien** : https://www.producthunt.com/products/minions/awards
- **Pourquoi c'est important** : Simplifie la gestion d'agents parallèles sans plonger dans du code d'orchestration.
- **Testable** : Oui — disponible sur Product Hunt (vérifier plan gratuit)

## 🔬 Research

### Deux modèles frontier ont réussi un test cyber-offensif complet en 32 étapes
- **Quoi** : Claude Mythos Preview, puis GPT-5.5 trois semaines plus tard, ont chacun passé un test de cyberattaque end-to-end en 32 étapes. L'UK AI Security Institute (AISI) estime que la capacité offensive des LLMs double tous les quatre mois.
- **Lien** : https://thehackernews.com/2026/05/2026-year-of-ai-assisted-attacks.html
- **Pourquoi c'est important** : Le seuil critique est désormais atteint : les modèles peuvent conduire des attaques complexes de bout en bout. Redéfinit la menace cyber et la politique de diffusion des modèles.
- **Testable** : Non

### Google DeepMind AI Co-Mathematician — FrontierMath Tier 4
- **Quoi** : L'AI Co-Mathematician de DeepMind a résolu 23/48 problèmes FrontierMath Tier 4 (benchmark de mathématiques de recherche avancée) et a contribué à la résolution de problèmes ouverts aux côtés de mathématiciens professionnels.
- **Lien** : https://hai.stanford.edu/news/inside-the-ai-index-12-takeaways-from-the-2026-report
- **Pourquoi c'est important** : Premier signe crédible qu'un modèle IA contribue à la recherche mathématique réelle (pas seulement aux concours IMO).
- **Testable** : Non — outil de recherche interne DeepMind

### TurboQuant (Google, ICLR 2026) — compression KV cache
- **Quoi** : Google présente TurboQuant à ICLR 2026, un algorithme réduisant significativement la mémoire consommée par le KV cache via PolarQuant (rotation vectorielle) et Quantized Johnson-Lindenstrauss compression.
- **Lien** : https://llm-stats.com/ai-news
- **Pourquoi c'est important** : Réduit le coût de serving des LLMs en inférence longue-context — impact direct sur la viabilité économique des agents.
- **Testable** : Non — paper de recherche (pas encore open-sourcé)
