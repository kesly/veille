## 🧠 Mises à jour de Modèles

### GPT-5.5 Instant — Nouveau modèle par défaut de ChatGPT
- **Quoi** : Remplace GPT-5.3 Instant comme modèle par défaut de ChatGPT — 52,5% de réduction des hallucinations sur des sujets critiques (médecine, droit, finance) en évaluation interne
- **Lien** : https://openai.com/index/gpt-5-5-instant/
- **Pourquoi c'est important** : La réduction des hallucinations est mesurée — pas une affirmation marketing. Améliore aussi la personnalisation et la clarté des réponses.
- **Testable** : Oui — disponible dans ChatGPT (tous plans) et API

### Kimi K2.6 — L'open-source qui bat GPT-5.5 en coding
- **Quoi** : Modèle open-weight de Moonshot AI, architecture multi-agent pouvant scaler jusqu'à 300 sous-agents parallèles pour les tâches longues — surpasse GPT-5.5 sur les benchmarks de code
- **Lien** : https://www.buildfastwithai.com/blogs/kimi-k2-6-review-benchmarks
- **Pourquoi c'est important** : Le meilleur modèle de coding est open-source et gratuit à héberger. Cursor Composer 2.5 est basé sur sa génération précédente (K2.5).
- **Testable** : Oui — open-weight, disponible sur Hugging Face (gratuit)

### Mistral Small 4 + Voxtral TTS
- **Quoi** : Mistral Small 4 (AA Intelligence Index 27.8, 256K ctx, raisonnement) disponible ; Voxtral TTS lancé fin mars — TTS open-source 3B-4B params, clonage vocal en 3s, 9 langues, bat ElevenLabs en blind tests
- **Lien** : https://venturebeat.com/orchestration/mistral-ai-just-released-a-text-to-speech-model-it-says-beats-elevenlabs-and
- **Pourquoi c'est important** : Voxtral tourne sur un seul GPU 16GB VRAM — TTS de qualité production accessible sans infrastructure cloud dédiée. Open-source.
- **Testable** : Oui — Voxtral TTS sur Hugging Face (gratuit, open-weights)
