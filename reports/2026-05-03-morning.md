## 🔥 Breaking

> Releases majeures, annonces qui changent la donne

### Claude Mythos Preview — Anthropic
- **Quoi** : Nouveau modèle frontier d'Anthropic en test limité, avec capacités de raisonnement et de code supérieures à l'humain
- **Lien** : https://red.anthropic.com/2026/mythos-preview/

Parallèlement, la Maison Blanche reprend contact avec Anthropic (exclue des contrats militaires du Pentagone) car ses modèles sont trop puissants pour être ignorés.
- **Testable** : Non — accès limité, pas encore public

### Pentagon AI Deals — 7 entreprises signent, Anthropic exclue
- **Quoi** : Le Département de la Défense américain a conclu des accords avec OpenAI, Google, Microsoft, Amazon AWS, SpaceX, Nvidia et Reflection pour déployer l'IA dans des réseaux classifiés

- **Pourquoi c'est important** : Anthropic a refusé de supprimer ses garde-fous de sécurité pour "toutes les utilisations légales" (incluant armes autonomes et surveillance de masse) — ce qui l'a exclue du contrat. Signal fort sur le débat sécurité vs. déploiement militaire de l'IA.
- **Testable** : Non

### Google AlphaEvolve — agent de découverte algorithmique
- **Quoi** : DeepMind présente AlphaEvolve, un agent qui combine Gemini avec un algorithme évolutionnaire pour découvrir et optimiser des algorithmes mathématiques inédits
- **Lien** : https://hai.stanford.edu/ai-index/2026-ai-index-report
- **Pourquoi c'est important** : AlphaEvolve retrouve la solution optimale dans 75 % des 50 problèmes de référence testés et surpasse les solutions existantes dans 20 % des cas. Un vrai pas vers l'IA capable de faire avancer la science de manière autonome.
- **Testable** : Non — pas encore d'accès public annoncé

## 🔬 Research

### Stanford HAI — AI Index 2026
- **Quoi** : Le rapport annuel Stanford sur l'état de l'IA révèle une accélération sans précédent : les benchmarks de coding sont passés de 60 % à quasi 100 % en un an ; les agents IA résolvent 77,3 % des tâches du monde réel (contre 20 % en 2025)
- **Lien** : https://hai.stanford.edu/ai-index/2026-ai-index-report
- **Pourquoi c'est important** : Les modèles dépassent maintenant 50 % sur "Humanity's Last Exam" (contre 8,8 % en 2025). Les agents en cybersécurité résolvent 93 % des problèmes (vs 15 % en 2024). La progression est exponentielle.
- **Testable** : Non — rapport de recherche

### MCP (Model Context Protocol) — 97 millions d'installs
- **Quoi** : Le protocole MCP d'Anthropic (donné à la Linux Foundation en décembre 2025) dépasse 97 millions d'installs en mars 2026. Python SDK mis à jour le 1er mai 2026.
- **Lien** : https://github.com/modelcontextprotocol
- **Pourquoi c'est important** : MCP est devenu le standard de facto pour connecter les agents aux APIs, fichiers et données externes. Tous les grands providers l'ont adopté. Support multimédia (images, vidéo, audio) en cours d'intégration.
- **Testable** : Oui — open-source
