# AI Watch — 2026-06-27 Matin

## 🔥 Breaking
> Releases majeures, annonces qui changent la donne

### OpenAI GPT-5.6 : Sol, Terra & Luna — accès ultra-restreint
- **Quoi** : OpenAI a lancé le 26 juin trois nouveaux modèles GPT-5.6 en preview limitée : Sol (flagship, le plus puissant jamais sorti), Terra (intermédiaire, 2x moins cher que GPT-5.5) et Luna (rapide et économique)
- **Lien** : https://openai.com/index/previewing-gpt-5-6-sol/
- **Pourquoi c'est important** : Sol est décrit comme le meilleur modèle OpenAI pour la cybersécurité et la biologie, avec un nouveau mode "ultra" utilisant des sous-agents pour les tâches complexes. Tarifs Sol : $5 input / $30 output pour 1M tokens. Mais seules ~20 entreprises approuvées par le gouvernement américain y ont accès pour l'instant.
- **Testable** : Non — preview partenaires uniquement, expansion prévue la semaine prochaine

### Anthropic proche d'un accord US pour débloquer Fable 5 & Mythos 5
- **Quoi** : Après deux semaines de blocage gouvernemental (depuis le 12 juin), Anthropic et l'administration Trump seraient sur le point de signer un accord pour lever les restrictions d'export sur ses deux modèles les plus puissants
- **Lien** : https://www.bloomberg.com/news/articles/2026-06-26/anthropic-moves-toward-deal-with-us-to-lift-curbs-on-ai-models
- **Pourquoi c'est important** : Les modèles avaient été bloqués après qu'un sénateur eut révélé que Mythos avait compromis "presque tous les systèmes classifiés US en quelques heures" lors d'un test. L'interdiction vise tous les ressortissants étrangers, y compris les employés d'Anthropic. La reprise d'accès à Fable 5 (leader SWE-bench à 95%) relancerait la compétition coding.
- **Testable** : Non — en attente de l'accord officiel

### ARD — Standard ouvert de découverte d'agents IA
- **Quoi** : Google, Microsoft, GitHub, Hugging Face, NVIDIA, Salesforce, Snowflake, Cisco et d'autres ont publié la spécification Agentic Resource Discovery (ARD) : un standard permettant aux agents IA de localiser, vérifier et connecter des outils, APIs et MCP servers à l'exécution
- **Lien** : https://developers.googleblog.com/announcing-the-agentic-resource-discovery-specification/
- **Pourquoi c'est important** : ARD est le "DNS pour agents IA". Il repose sur un fichier `ai-catalog.json` hébergé sur votre domaine et une API de registre fédéré. GitHub Copilot l'utilise déjà via son nouvel "Agent Finder". Licencié Apache 2.0 — adoption encore faible mais coalition de poids.
- **Testable** : Oui — spec disponible sur https://agenticresourcediscovery.org/ + implémentation de référence GitHub

### Mistral OCR 4 — IA documentaire on-premise
- **Quoi** : Mistral a publié le 23 juin OCR 4, un modèle d'intelligence documentaire conscient de la structure (tableaux, layouts) déployable entièrement sur infrastructure privée
