## 🔥 Breaking
> Releases majeures, annonces qui changent la donne

### Anthropic + SpaceX : accord compute massif
- **Quoi** : Anthropic accède à la totalité de Colossus 1 (Memphis) — 220 000+ GPU Nvidia, 300 MW de capacité disponibles en moins d'un mois
- **Lien** : https://www.bloomberg.com/news/articles/2026-05-06/anthropic-inks-computing-deal-with-spacex-to-meet-ai-demand
- **Pourquoi c'est important** : Réponse directe à la croissance 80x du T1 2026 qui avait saturé les infras et dégradé la fiabilité de Claude Pro/Max. Dario Amodei avait prévu 10x de croissance... L'accord double immédiatement les limites de débit sur Claude Code (Pro, Max, Team, Enterprise) et supprime la réduction d'heures de pointe.
- **Testable** : Oui — limites Claude Code augmentées dès maintenant

### Anthropic Project Glasswing : Claude Mythos Preview en accès restreint
- **Quoi** : Anthropic donne accès à son modèle frontier inédit Claude Mythos Preview à un cercle fermé (AWS, Apple, Cisco, Google, JPMorgan Chase, Microsoft) pour détecter des vulnérabilités logicielles critiques avant les acteurs malveillants
- **Lien** : https://www.cnbc.com/2026/05/05/anthropic-ceo-cyber-moment-of-danger-mythos-vulnerabilities.html
- **Pourquoi c'est important** : Mythos mène le classement GPQA Diamond (94,6%). Dario Amodei parle d'un "moment de danger cyber" — le modèle a déjà trouvé des dizaines de milliers de vulnérabilités décennales dans des logiciels critiques. Anthropic positionne Mythos dans une classe "restricted" à ne pas diffuser publiquement.
- **Testable** : Non — accès restreint aux partenaires sélectionnés

### OpenAI lève 122 milliards de dollars
- **Quoi** : OpenAI annonce une levée de fonds de 122 milliards de dollars pour accélérer la prochaine phase de l'IA
- **Lien** : https://openai.com/index/accelerating-the-next-phase-ai/
- **Pourquoi c'est important** : La plus grande levée de fonds de l'histoire de la tech. Consolide la position d'OpenAI dans la course aux infrastructures face à Anthropic et Google.
- **Testable** : Non

### Anthropic dépasse OpenAI en ARR
- **Quoi** : L'ARR d'Anthropic a dépassé celui d'OpenAI pour la première fois, atteignant 30 milliards de dollars annualisés contre 24 milliards pour OpenAI (progression de 9 → 30 Md$ en 4 mois)
- **Lien** : https://news.ycombinator.com/item?id=47817581

## 🆕 Nouveaux Outils & Produits

### OpenAI : 3 modèles voix multilingues + pub ChatGPT
- **Quoi** : OpenAI lance trois modèles voix capable de traduire 70 langues en temps réel ; Zillow annonce +26 points de taux de succès d'appel en test. En parallèle, ChatGPT ouvre un Ads Manager en self-serve pour les petites entreprises.

- **Testable** : Oui — modèles voix via l'API OpenAI (accès progressif)

### Mistral Voxtral TTS : premier TTS open-weight frontier
- **Quoi** : Mistral publie Voxtral TTS, un modèle text-to-speech open-weight de qualité frontier conçu pour l'entreprise, qu'ils décrivent comme surpassant ElevenLabs. Complète un pipeline speech-to-speech complet déployable en self-hosted.
- **Lien** : https://venturebeat.com/orchestration/mistral-ai-just-released-a-text-to-speech-model-it-says-beats-elevenlabs-and
- **Pourquoi c'est important** : Premier modèle TTS de qualité frontier à poids ouverts. Les entreprises peuvent déployer sans dépendance à un fournisseur externe.
- **Testable** : Oui — poids disponibles, auto-hébergeable

### AWS MCP Server : disponibilité générale
- **Quoi** : AWS annonce la disponibilité générale de l'AWS MCP Server — un serveur MCP managé permettant aux agents de coder de façon sécurisée avec n'importe quelle API AWS, avec guardrails IAM, logs CloudTrail et métriques CloudWatch
- **Lien** : https://aws.amazon.com/about-aws/whats-new/2026/05/aws-mcp-server/
- **Pourquoi c'est important** : Toutes les APIs AWS deviennent accessibles à un agent via un seul outil MCP, avec exécution Python sandbox. Première solution MCP cloud enterprise de cette envergure.
- **Testable** : Oui — disponible sur AWS console

### Runway Characters : vidéo parlante depuis une image
- **Quoi** : Runway lance Characters, une feature qui génère une vidéo animée (parole, clignement, mouvement de tête) à 24 fps depuis une seule image, avec 37ms de traitement par frame
- **Lien** : https://www.theaimarketers.ai/news050726/
- **Pourquoi c'est important** : Supprime le besoin de tournage pour les avatars vidéo. Cas d'usage : e-learning, publicité, UX produit.
- **Testable** : Oui — via runway.ml

### Anthropic : agents financiers prépackagés + Opus 4.7
- **Quoi** : Anthropic lance des templates d'agents IA prêts-à-l'emploi pour la finance (banques, trading, analyse), des add-ins M365 (Excel, PowerPoint, Word, Outlook) et un connecteur Moody's MCP. Claude Opus 4.7, optimisé finance, est disponible.

## 🔬 Research

### GDPval : nouveau benchmark OpenAI sur la valeur économique
- **Quoi** : OpenAI publie GDPval, un benchmark couvrant 44 métiers de 9 industries clés du PIB américain (développeurs, avocats, infirmières, ingénieurs mécaniques…) pour mesurer la capacité des LLMs à effectuer du vrai travail de connaissance
- **Lien** : https://hai.stanford.edu/ai-index/2026-ai-index-report/technical-performance
- **Pourquoi c'est important** : Premier benchmark aligné sur l'impact économique réel plutôt que sur des tâches académiques abstraites. Devient la référence pour justifier les déploiements enterprise.
- **Testable** : Non — benchmark de recherche

### Humanity's Last Exam : +30 points en un an
- **Quoi** : Les modèles frontier ont progressé de 30 points de pourcentage en un an sur HLE, un benchmark conçu pour être difficile pour les IA et favorable aux experts humains
- **Lien** : https://lmcouncil.ai/benchmarks
- **Pourquoi c'est important** : Rythme de progression sans précédent sur un benchmark "AGI-level". Soulève la question de la saturation imminente des métriques d'évaluation.
- **Testable** : Non

### Crise de crédibilité des benchmarks
- **Quoi** : La contamination des données d'entraînement affecte de plus en plus les benchmarks standards — le consensus en mai 2026 est que les leaderboards publics sont peu fiables
- **Lien** : https://lmcouncil.ai/benchmarks
- **Pourquoi c'est important** : Rend difficile la comparaison objective des modèles. Crée un besoin urgent de benchmarks privés et vérifiables.
- **Testable** : Non
