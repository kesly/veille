# AI Watch — 2026-04-24 Matin

## 🔥 Breaking
> Releases majeures, annonces qui changent la donne

### GPT-5.5 — OpenAI
- **Quoi** : Nouveau modèle frontier d'OpenAI centré sur le coding et le raisonnement autonome, sorti le 23 avril, 6 semaines après GPT-5.4
- **Lien** : https://openai.com/index/introducing-gpt-5-5/
- **Pourquoi c'est important** : 88,7 % sur SWE-bench, 92,4 % sur MMLU, 60 % moins d'hallucinations que GPT-5.4, 39,6 % sur FrontierMath Tier 4 (quasi le double de Claude Opus 4.7). Modèle "agentic" capable de switcher entre outils pour finir des tâches complexes. Tarif : $5/1M tokens input, $30/1M output ($30/$180 pour la version Pro).
- **Testable** : Oui — via ChatGPT (plans Plus, Pro, Business, Enterprise) et Codex

### DeepSeek V4 — Open Source Apache 2.0
- **Quoi** : DeepSeek publie ce jour V4-Pro (1,6T paramètres) et V4-Flash (284B), avec contexte 1M tokens, sous licence Apache 2.0
- **Lien** : https://huggingface.co/deepseek-ai/DeepSeek-V4-Pro
- **Pourquoi c'est important** : Meilleur modèle open source sur l'agentic coding, surpasse tous les modèles open source en math, STEM et coding compétitif. Architecture hybride CSA+HCA inédite. Disponible poids libres sur Hugging Face dès aujourd'hui.
- **Testable** : Oui — poids téléchargeables sur Hugging Face, API disponible sur ofox.ai

### Google Cloud Next 2026 — Vague d'annonces
- **Quoi** : Google dévoile la 8e génération de TPUs (deux puces spécialisées, superpod de 9 600 TPUs), la Gemini Enterprise Agent Platform, et un fonds de 750 M$ pour les partenaires agentic
- **Lien** : https://cloud.google.com/blog/topics/google-cloud-next/welcome-to-google-cloud-next26
- **Pourquoi c'est important** : TPU 8t pour l'entraînement scale jusqu'à 2 pétaoctets de mémoire partagée. La Gemini Enterprise Agent Platform inclut un Agent Designer, une boîte de réception d'agents, et des Skills. Virgo Network = nouveau fabric datacenter megascale. Infrastructure cloud taillée pour l'ère agentique.
- **Testable** : Oui — Veo 3.1 Lite gratuit (10 vidéos/mois) via vids.new ou labs.google/flow avec un compte Google standard

## 🆕 Nouveaux Outils & Produits

### MCP Apps — Première extension officielle MCP en production
- **Quoi** : MCP Apps, l'extension officielle du Model Context Protocol, passe en production avec support UI interactive (dashboards, formulaires, visualisations dans le chat)
- **Lien** : https://blog.premai.io/25-best-mcp-servers-for-ai-agents-complete-setup-guide-2026/
- **Pourquoi c'est important** : Adoptée par Claude, ChatGPT, VS Code + GitHub Copilot, Goose, Postman. Les tool calls peuvent retourner des composants UI directement dans la conversation agent. L'écosystème MCP dépasse 10 000 serveurs publics actifs et 164M téléchargements/mois du SDK Python.
- **Testable** : Oui — compatible Claude, ChatGPT et VS Code déjà aujourd'hui

### xAI Grok STT + TTS APIs
- **Quoi** : xAI lance deux APIs audio standalone : Speech-to-Text (batch + streaming, 25 langues) et Text-to-Speech (5 voix expressives, 20 langues, balises `[laugh]`, `<whisper>`)
- **Lien** : https://www.marktechpost.com/2026/04/18/xai-launches-standalone-grok-speech-to-text-and-text-to-speech-apis-targeting-enterprise-voice-developers/
- **Pourquoi c'est important** : 5,0 % d'erreur sur les benchmarks téléphoniques (vs 12 % ElevenLabs, 21,3 % AssemblyAI). Normalisation de texte inverse et diarisation intégrées. Déjà dans Grok Voice, Tesla et Starlink. TTS : $4,20/1M caractères.
- **Testable** : Oui — API disponible via la console xAI (entreprises)

### OpenAI Agents SDK — Mise à jour majeure
- **Quoi** : OpenAI a livré début avril une mise à jour majeure de l'Agents SDK : sandboxing, harness long-horizon, orchestration de sous-agents, support provider-agnostic (100+ LLMs)
- **Lien** : https://github.com/lastmile-ai/mcp-agent
- **Pourquoi c'est important** : Les agents peuvent maintenant opérer dans des espaces de travail isolés pour des tâches multi-étapes complexes, tout en supportant des LLMs tiers. Franchi le seuil de la production pour les workflows entreprise.
- **Testable** : Oui — open source, disponible sur PyPI

### Cursor — Valorisation $50B, acquisition potentielle par SpaceX
- **Quoi** : Cursor (ARR $2B) était en discussions pour lever $2B avec a16z, Thrive et Nvidia — suspendu après une offre de collaboration de SpaceX à $10B + rachat possible à $60B
- **Lien** : https://techcrunch.com/2026/04/22/how-spacex-preempted-a-2b-fundraise-with-a-60b-buyout-offer/
- **Pourquoi c'est important** : La saga Cursor/SpaceX est le signe le plus clair que les outils de coding AI sont devenus des actifs stratégiques de premier ordre. Une acquisition à $60B serait l'une des plus grandes de l'histoire tech.
- **Testable** : Non (deal en cours)

## 📝 À surveiller
> Tendances émergentes, betas, rumeurs crédibles

- **Project Prometheus (Bezos)** : Physical AI lab levant $10B à $38B de valorisation, backé par BlackRock et JPMorgan. Focalisé sur l'IA pour la fabrication, l'aérospatiale et la robotique. À suivre de près — Bezos reprend un rôle opérationnel pour la première fois depuis Amazon.

- **SpaceX / Cursor** : Si l'acquisition à $60B se concrétise, ce serait le signal que les éditeurs de logiciels traditionnels (et même l'aérospatiale) voient les coding agents comme des actifs critiques.
- **OpenAI ChatGPT for Clinicians** : Version clinique gratuite pour les médecins US vérifiés (HIPAA, CME, workflows réutilisables). Première incursion sérieuse d'OpenAI dans la santé en accès direct praticien.
- **GPT-5.4-Cyber** : OpenAI brief en privé les agences fédérales US, gouvernements d'État et alliés Five Eyes sur un modèle spécialisé cybersécurité — non public pour l'instant.
- **A2A + MCP convergence** : Les deux protocoles (A2A pour agent↔agent, MCP pour agent↔outil) semblent vouloir coexister plutôt que se fusionner — l'écosystème se fragmente légèrement.
