### OpenAI — GPT-5.5 Instant + nouveaux modèles temps réel

- **Quoi** : Anthropic ouvre l'accès public élargi à ses agents autonomes capables de gérer des workflows longs (coding, finance, droit), avec coordination de sous-agents et évaluation par rubrique

## 🆕 Nouveaux Outils & Produits

### OpenHuman — Agent desktop open-source #1 sur GitHub Trending
- **Quoi** : Agent de bureau open-source (tinyhumansai) qui lit le contexte utilisateur avant toute interaction — intégrations OAuth one-click (Gmail, Notion, GitHub, Slack, Linear, Jira…), mémoire locale en Markdown, 118+ connecteurs
- **Lien** : https://github.com/tinyhumansai/openhuman
- **Pourquoi c'est important** : Se différencie d'OpenClaw/Hermes en partant du profil utilisateur plutôt que de la tâche — 7,8k stars en beta, 629 forks, propulsé #1 GitHub Trending le 16 mai
- **Testable** : Oui — open-source, installation desktop (v0.53.43, beta active)

### AWS MCP Server — Disponibilité générale
- **Quoi** : AWS lance la GA de son serveur MCP qui donne aux agents de coding un accès auditable à tous les services AWS via IAM + CloudWatch + CloudTrail ; inclut sandbox Python pour scripts multi-étapes
- **Lien** : https://aws.amazon.com/about-aws/whats-new/2026/05/aws-mcp-server/
- **Pourquoi c'est important** : Infrastructure MCP enterprise-grade clé en main — les agents peuvent appeler n'importe quelle API AWS sans accès shell local
- **Testable** : Oui — AWS Console, compatible Claude Code / Cursor / Copilot

### GitHub Copilot → AI Credits (1er juin 2026)
- **Quoi** : GitHub bascule Copilot en facturation à l'usage (AI Credits) le 1er juin : Pro à $10/mois, Pro+ à $39/mois, nouveau plan Max à $100/mois
- **Lien** : https://github.blog/news-insights/company-news/github-copilot-is-moving-to-usage-based-billing/
- **Pourquoi c'est important** : Fin du forfait "illimité" — les gros utilisateurs (Copilot Chat, agent mode, longs contextes) vont payer plus ; à anticiper pour les équipes
- **Testable** : Non — effectif le 1er juin

### Google Veo 3.1 Lite — Vidéo IA à $0,05/s
- **Quoi** : Modèle de génération vidéo text-to-video et image-to-video, 720p/1080p, audio synchronisé (effets, dialogues, musique), <50% du coût de Veo 3.1 Fast
- **Lien** : https://blog.google/innovation-and-ai/technology/ai/veo-3-1-lite/
- **Pourquoi c'est important** : Point d'entrée le plus abordable de Google pour la vidéo générée — viable pour les pipelines à fort volume (A/B testing, automatisation)
- **Testable** : Oui — Gemini API + Google AI Studio ($0,05/s en 720p)

## 🧠 Mises à jour de Modèles

### Mistral Medium 3.5 128B — Open weights, remplace 3 modèles
- **Quoi** : Modèle dense 128B sous licence MIT modifiée, fenêtre 256k tokens, multimodal, reasoning configurable — remplace Medium 3.1, Magistral et Devstral 2 en un seul modèle
- **Lien** : https://huggingface.co/mistralai/Mistral-Medium-3.5-128B
- **Pourquoi c'est important** : SWE-Bench Verified à 77,6% (coding réel sur GitHub issues) + agent Work Mode dans Le Chat qui ouvre des PRs automatiquement ; $1,50/$7,50 par million de tokens
- **Testable** : Oui — Hugging Face (self-host) ou API Mistral / Le Chat

### Meta Muse Spark — Premier modèle des Meta Superintelligence Labs
- **Quoi** : Modèle multimodal de la nouvelle série Muse (code "Avocado"), 3 modes : Instant / Thinking / Contemplating (multi-agents en parallèle) ; équivalent Llama 4 Maverick à 10x moins de compute
- **Lien** : https://ai.meta.com/blog/introducing-muse-spark-msl/
- **Pourquoi c'est important** : Premier résultat concret de l'arrivée d'Alexandr Wang chez Meta — nouveau paradigme de scaling en série plutôt qu'en saut géant
- **Testable** : Oui — Meta AI web + app (déploiement WhatsApp/Instagram/Facebook en cours)

### Mistral Voxtral TTS — TTS open weights bat ElevenLabs
- **Quoi** : Modèle TTS 4B paramètres, 9 langues, clonage vocal à partir de 3 secondes de référence, architecture hybride autorégressif + flow-matching
- **Lien** : https://mistral.ai/news/voxtral-tts
- **Pourquoi c'est important** : Taux de victoire de 87,8% vs ElevenLabs Flash v2.5 en espagnol, 79,8% en hindi ; open weights auto-hébergeable + API cloud à $0,016/1000 chars
- **Testable** : Oui — Hugging Face (gratuit, self-host) ou API Mistral

### Labs chinois — Vague de modèles coding open weights
- **Quoi** : Quatre releases quasi-simultanées : GLM-5.1 (Z.ai, MoE 744B/40B actifs), MiniMax M2.7, Kimi K2.6 (Moonshot), DeepSeek V4 (MoE 1,6T/49B actifs)
- **Lien** : https://llm-stats.com/ai-news
- **Pourquoi c'est important** : Pression continue sur les prix et les benchmarks coding ; DeepSeek V4 dépasse les 1T de paramètres en MoE actif
- **Testable** : Oui — modèles disponibles sur Hugging Face et via leurs APIs respectives
