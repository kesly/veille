## 🔥 Breaking

### Claude Fable 5 & Mythos 5 — Retour mondial confirmé
- **Quoi** : Anthropic a restauré l'accès global à Fable 5 et Mythos 5 le 1er juillet, après suspension forcée le 12 juin due aux export controls US imposés à ces modèles frontier.
- **Lien** : https://www.anthropic.com/news/redeploying-fable-5
- **Pourquoi c'est important** : Le retour inclut une limite d'usage (50% du quota hebdomadaire pour les plans Pro/Max/Team jusqu'au 7 juillet), mais signale la levée des restrictions US sur ces modèles — Fable 5 reste le modèle le plus puissant d'Anthropic (80,3% SWE-Bench Pro, fenêtre de 1M tokens).
- **Testable** : Oui — Claude.ai, Claude Code, Claude Cowork (plans Pro, Max, Team, Enterprise select)

### Claude Sonnet 5 — Nouveau modèle par défaut Anthropic
- **Quoi** : Lancé le 30 juin, Sonnet 5 est maintenant le modèle par défaut des plans Free et Pro ; tarif d'introduction à $2/M tokens input et $10/M output jusqu'au 31 août.
- **Lien** : https://www.anthropic.com/news/claude-sonnet-5
- **Pourquoi c'est important** : 63,2% SWE-Bench Pro, fenêtre 1M tokens, 128k tokens max output, thinking adaptatif — conçu pour les workflows agentiques multi-étapes sans supervision. Il remplace Sonnet 4.6 comme baseline "abordable et puissant".
- **Testable** : Oui — API Anthropic et Claude.ai dès maintenant

### MCP Spec 2026-07-28 — Release Candidate disponible
- **Quoi** : La plus grande révision du Model Context Protocol depuis son lancement, avec SDK betas en Python, TypeScript, Go et C# désormais disponibles.
- **Lien** : https://blog.modelcontextprotocol.io/posts/2026-07-28-release-candidate/
- **Pourquoi c'est important** : Passage à une architecture stateless (plus besoin de sticky sessions), authentification OAuth/OIDC standardisée, caching via `ttlMs`, et ajout des extensions MCP Apps (UI server-side) et Tasks (travaux longue durée). Brise la compatibilité ascendante : le handshake `initialize/initialized` est supprimé.
- **Testable** : Oui — SDK betas open source disponibles sur GitHub
