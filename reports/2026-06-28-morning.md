# AI Watch — 2026-06-28 Matin

## 🔥 Breaking

### GPT-5.6 Sol / Terra / Luna — Preview limité sous supervision gouvernementale
- **Quoi** : OpenAI a lancé le 26 juin une suite de trois modèles — Sol (flagship), Terra (polyvalent), Luna (rapide/économique) — en accès restreint à ~20 entreprises approuvées par le gouvernement américain
- **Lien** : https://openai.com/index/previewing-gpt-5-6-sol/
- **Pourquoi c'est important** : Sol cible les problèmes les plus difficiles (cybersécurité, code complexe) avec la stack de sécurité la plus robuste à ce jour. Pricing : Sol $5/$30, Terra $2,50/$15, Luna $1/$6 par million de tokens. C'est la première fois qu'OpenAI sort un modèle sous restriction gouvernementale explicite.
- **Testable** : Non — preview limité aux partenaires approuvés par le gouvernement. Disponibilité générale prévue "dans les prochaines semaines"

### Claude Fable 5 & Mythos 5 — Suspendus par l'US Government, accès partiel restauré
- **Quoi** : Lancés le 9 juin, Fable 5 et Mythos 5 ont été suspendus mondialement le 12 juin suite à une directive de contrôle d'export du Department of Commerce. Le 26 juin, l'administration Trump a autorisé un accès limité à ~100 entreprises et agences fédérales
- **Lien** : https://www.cnbc.com/2026/06/26/us-government-anthropic-claude-mythos5-ai.html
- **Pourquoi c'est important** : Premier cas de suspension mondiale forcée d'un modèle IA par le gouvernement américain. La raison invoquée : une technique de jailbreak permettant de contourner les garde-fous cybersécurité de Mythos 5. Anthropic estimait la faille étroite. Cela crée un précédent majeur sur la gouvernance des modèles frontière.
- **Testable** : Non — accès réservé aux organisations approuvées ; Claude Opus 4.8 reste disponible et est actuellement #1 sur l'Artificial Analysis Intelligence Index (61,4)

## 🆕 Nouveaux Outils & Produits

### Cursor 3 "Glass" — IDE Agent-First avec agents cloud parallèles
- **Quoi** : Cursor a sorti en juin des mises à jour majeures de son redesign "Glass" (lancé le 2 avril), ajoutant `/in-cloud` pour lancer des sous-agents dans des VMs isolées avec snapshots réutilisables, et un Agents Window comme surface principale
- **Lien** : https://releasebot.io/updates/cursor
- **Pourquoi c'est important** : Passage d'un IDE classique à un "command center" multi-agents — on peut exécuter des tâches parallèles dans des branches isolées sans toucher le workspace local. Compétiteur direct à GitHub Copilot Workspaces et Devin.
- **Testable** : Oui — Cursor Pro ; lancer `/in-cloud` dans l'Agents Window

### Devin Desktop (ex-Windsurf) — Agent Command Center Kanban
- **Quoi** : Cognition a rebrandé Windsurf en Devin Desktop le 2 juin, avec un tableau de bord Kanban "Agent Command Center" comme vue par défaut, et "Spaces" pour le partage de contexte entre agents sur un même projet
- **Lien** : https://cognition.com/blog/windsurf
- **Pourquoi c'est important** : Windsurf (500 000+ utilisateurs) fusionne avec Devin dans une interface orientée orchestration multi-agents. Les plans et extensions existants migrent automatiquement. Devin Cloud passe du plan enterprise au plan Pro.
- **Testable** : Oui — si vous utilisiez Windsurf, la mise à jour est automatique (OTA). Sinon : devin.ai

### Grok Imagine Video 1.5 — Génération vidéo 2x plus rapide
- **Quoi** : xAI a lancé le 17 juin Grok Imagine Video 1.5, avec génération d'une vidéo 6 secondes en 720p en ~25 secondes (contre 40+ secondes avant)
- **Lien** : https://x.ai/news
- **Pourquoi c'est important** : Amélioration significative de la vitesse pour la génération vidéo côté xAI. Positionnement face à Kling 3.0 et Veo 3.1.
- **Testable** : Oui — via Grok sur X (anciennement Twitter)

### Grok STT & TTS APIs — Speech d'entreprise par xAI

### Microsoft Scout — Premier "Autopilot" avec identité Entra
- **Quoi** : Microsoft a lancé Scout, un agent autonome "always-on" avec sa propre identité Entra, lors de Build 2026 (2-3 juin), accompagné des Work IQ APIs avec outils MCP génériques

### Grok 4.3 — Sur AWS avec 1M tokens de contexte
- **Quoi** : Grok 4.3 est maintenant disponible pour les développeurs AWS dans les régions supportées, avec fenêtre de contexte 1M tokens et modes de raisonnement configurables (none/low/medium/high)

## 🔬 Research

### Attaque de distillation Anthropic/Qwen — 25 000 comptes frauduleux, 28,8M échanges
- **Quoi** : Anthropic a adressé une lettre au Sénat américain révélant une attaque de distillation massive : des opérateurs affiliés à Alibaba/Qwen ont créé ~25 000 comptes frauduleux et généré 28,8 millions d'échanges avec Claude entre le 22 avril et le 5 juin 2026
- **Lien** : https://www.buildfastwithai.com/blogs/ai-news-today-june-26-2026
- **Pourquoi c'est important** : Première attaque de distillation documentée à cette échelle. Soulève des questions sur les protections anti-distillation des providers et sur la gouvernance des APIs publiques.
- **Testable** : Non — incident de sécurité

### OpenClaw — Projet open source viral : 210 000 stars GitHub
- **Quoi** : OpenClaw (assistant IA personnel, auto-hébergé, intégrant WhatsApp/Telegram/Slack/Discord/iMessage via 50+ MCP integrations) a explosé de 9K à 210K+ stars depuis janvier 2026
- **Lien** : https://github.com/trending
- **Pourquoi c'est important** : Alternative open source aux assistants cloud, tout tourne localement. Représente la tendance "AI sur ses propres appareils" qui monte en 2026.
- **Testable** : Oui — open source, auto-hébergeable

### MCP : 1 milliard d'appels/mois, Streamable HTTP nouveau standard
- **Quoi** : Le protocole MCP atteint 1 milliard d'appels outils/mois chez Claude seul ; SSE est maintenant déprécié, le standard est Streamable HTTP pour les serveurs distants
- **Lien** : https://workos.com/blog/everything-your-team-needs-to-know-about-mcp-in-2026
- **Pourquoi c'est important** : MCP est devenu l'USB-C de l'IA. 10 000+ serveurs publics, 97M téléchargements/mois des SDKs. Si vous construisez des outils IA, MCP est incontournable.
- **Testable** : Oui — spec disponible, SDKs Python/TypeScript
