## 🔥 Breaking

> Releases majeures, annonces qui changent la donne

### xAI lance Grok 4.3 avec clonage vocal et APIs STT/TTS
- **Quoi** : Grok 4.3 sort avec une réduction de prix de 40%, 1M tokens de contexte, vidéo native, et une suite complète de clonage vocal
- **Lien** : https://x.ai/news/grok-stt-and-tts-apis et https://venturebeat.com/technology/xai-launches-grok-4-3-at-an-aggressively-low-price-and-a-new-fast-powerful-voice-cloning-suite
- **Pourquoi c'est important** : $1.25/M tokens en input (vs GPT-5.5 et Opus 4.7 bien plus chers), 1M tokens de contexte, entrée vidéo native. Le clonage vocal crée un modèle de voix production-ready depuis 60s de parole, avec 80+ voix preset dans 28 langues. Les APIs STT/TTS sont désormais disponibles pour développeurs (même stack que Tesla/Starlink support).
- **Testable** : Oui — via xAI Console ; clonage vocal gratuit avec Grok 4.3

### Claude Opus 4.7 disponible + Claude Mythos en preview limitée
- **Quoi** : Opus 4.7 passe en GA avec des gains majeurs sur les tâches d'ingénierie logicielle difficiles ; Mythos révèle des capacités de hacking au niveau des meilleurs humains
- **Lien** : https://www.anthropic.com/news/claude-opus-4-7 et https://red.anthropic.com/2026/mythos-preview/
- **Pourquoi c'est important** : Mythos est le premier modèle d'Anthropic ayant trouvé des milliers de vulnérabilités critiques dans chaque OS et navigateur majeur. Il est distribué via le Project Glasswing uniquement à des partenaires industriels critiques et développeurs open-source sélectionnés — signal fort sur les capacités offensives de la nouvelle génération de modèles.
- **Testable** : Opus 4.7 : Oui (API Anthropic). Mythos : Non (accès restreint via Project Glasswing)

### Claude Code /ultrareview — revue de PR par agents cloud en parallèle
- **Quoi** : Feature `/ultrareview` disponible en research preview pour Team/Enterprise, orchestrant plusieurs agents reviewers en parallèle depuis le cloud
- **Lien** : https://releasebot.io/updates/anthropic/claude-code
- **Pourquoi c'est important** : Chaque finding est reproduit indépendamment avant d'être remonté — élimine les faux positifs. L'agent explore le code depuis plusieurs angles (logique, edge cases, sécu, perf) sans intervention humaine. Marque un changement dans la façon dont la revue de code est effectuée.
- **Testable** : Oui — 3 runs gratuits pour Team/Enterprise ; `claude /ultrareview` dans le CLI

## 📝 À surveiller

> Tendances émergentes, betas, rumeurs crédibles

- **OpenAI à $852Md de valorisation** : Le tour de table de $122Md clôturé fin mars (Amazon $50Md, Nvidia $30Md, SoftBank $30Md) est le plus grand de l'histoire du private equity tech. Une IPO OpenAI reste conditionnée à l'atteinte de l'AGI selon les termes Amazon — signal fort sur la direction stratégique.
- **MCP à 97M d'installs** : Le Model Context Protocol d'Anthropic est devenu le standard de facto pour connecter les agents aux outils externes. La roadmap 2026 inclut transport HTTP stateless, Tasks asynchrones et gouvernance enterprise. À suivre : quels nouveaux MCP servers sortent chaque semaine.
- **Claude Mythos & Project Glasswing** : La preview interne montre des capacités offensives inédites. Anthropic joue la transparence (publication sur red.anthropic.com) mais le déploiement restera restreint — à suivre pour les implications sur la régulation IA.
- **OpenClaw** — 210k+ étoiles GitHub, le repo open-source IA à la croissance la plus rapide de l'histoire. À surveiller pour comprendre l'émergence du prochain paradigme.
- **Restructuration Microsoft-OpenAI** : La fin de l'exclusivité Azure pour OpenAI, qui peut désormais servir via AWS et Google Cloud, redistribue les cartes dans le cloud IA. Impact sur les prix et la disponibilité des modèles OpenAI à venir.
- **Novo Nordisk + OpenAI** : Partenariat stratégique pour intégrer l'IA dans la découverte de médicaments, les essais cliniques et la supply chain d'ici fin 2026 — modèle à répliquer dans d'autres verticales pharma.
