### Mistral Voxtral TTS
- **Quoi** : Premier modèle TTS de Mistral AI, 4 milliards de paramètres, supporte 9 langues (anglais, français, allemand, espagnol, néerlandais, portugais, italien, hindi, arabe).
- **Lien** : https://mistral.ai/news/voxtral-tts
- **Pourquoi c'est important** : Léger, multilingue, avec le français comme langue de premier ordre — bon choix pour des agents voix francophones à faible coût.
- **Testable** : Oui — via API Mistral (payant).

### Inworld Realtime TTS-2
- **Quoi** : Modèle voix temps réel en boucle fermée : il prend en entrée l'audio des tours précédents (pas seulement la transcription) — il "entend" comment vous parliez. Latence médiane < 200ms sur WebSocket persistant.
- **Lien** : https://www.marktechpost.com/2026/05/05/inworld-ai-launches-realtime-tts-2-a-closed-loop-voice-model-that-adapts-to-how-you-actually-talk/
- **Pourquoi c'est important** : Adaptation du ton et du style en temps réel selon la prosodie de l'utilisateur — un bond qualitatif pour les agents conversationnels.
- **Testable** : Oui — via Inworld API (preview).

### GPT-5.5 Instant (OpenAI, 5 mai 2026)
- **Quoi** : Nouveau modèle par défaut de ChatGPT (tiers gratuit et payant), remplace GPT-5.3 Instant. Hallucinations réduites, meilleure personnalisation.
- **Lien** : https://openai.com/index/gpt-5-5-instant/
- **Pourquoi c'est important** : C'est le modèle que des centaines de millions d'utilisateurs ChatGPT utilisent maintenant. Amélioration notable en droit, médecine, finance.
- **Testable** : Oui — ChatGPT (gratuit).

### SubQ 1M-Preview (Subquadratic, 5 mai 2026)
- **Quoi** : Premier LLM commercial basé sur une architecture d'attention entièrement subquadratique (pas un transformer standard), avec une fenêtre de contexte native de 12 millions de tokens.
- **Lien** : https://llm-stats.com/llm-updates
- **Pourquoi c'est important** : ~1/5e du coût des modèles frontier, jusqu'à 52x plus rapide sur l'attention à grande échelle. Architecture potentiellement disruptive si les performances suivent.
- **Testable** : Oui — via API SubQ (preview payante).
