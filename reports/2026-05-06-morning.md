## 🔥 Breaking

> Releases majeures, annonces qui changent la donne

### GPT-5.5 Instant — Nouveau modèle par défaut de ChatGPT
- **Quoi** : OpenAI remplace GPT-5.3 Instant par GPT-5.5 Instant comme modèle par défaut de ChatGPT
- **Lien** : https://openai.com/index/gpt-5-5-instant/
- **Pourquoi c'est important** : 52,5 % moins d'hallucinations sur les requêtes à enjeux élevés (médecine, droit, finance), 1 million de tokens de contexte via API, 400K via Codex. Scores : 82,7 % sur Terminal-Bench 2.0 (vs 75,1 % pour GPT-5.4), 81,2 sur AIME 2025. Réponses 30 % plus courtes et plus directes.
- **Testable** : Oui — disponible immédiatement sur ChatGPT (gratuit et payant) et via API OpenAI

### Anthropic : Claude Opus 4.7 + 10 agents financiers
- **Quoi** : Anthropic lance officiellement Claude Opus 4.7 (GA) accompagné d'une bibliothèque de ~10 agents pré-construits pour la finance, une intégration Microsoft 365 et un partenariat avec Moody's
- **Lien** : https://www.anthropic.com/news/finance-agents
- **Pourquoi c'est important** : Les agents couvrent pitchbooks, analyse de crédit, KYC, underwriting, clôture de fin de mois et audit de contrats d'assurance. Moody's intègre son moteur (600M+ entreprises) directement dans Claude. Intégration native dans Excel, Word, PowerPoint (Outlook bientôt). Score de 64,4 % sur le benchmark Finance Agent — premier du classement. Goldman Sachs + Blackstone lancent une société de services IA basée sur Claude pour les ETI.
- **Testable** : Oui — Claude Enterprise + via API Anthropic et Amazon Bedrock

### DeepSeek V4 Pro — Modèle open-source à 1,6T paramètres qui rivalise avec les frontières fermées
- **Quoi** : DeepSeek publie V4-Pro (1,6T params MoE, 49B activés) et V4-Flash (284B, 13B activés) — deux modèles open-weights avec contexte 1M tokens
- **Lien** : https://api-docs.deepseek.com/news/news260424
- **Pourquoi c'est important** : SWE-bench Verified à 80,6 % (quasi-égal à Claude Opus 4.7 à 80,8 %), Codeforces ELO 3 206 (devant GPT-5.5 à 3 168), LiveCodeBench 93,5. Prix : $0,145/MTok en entrée, soit ~7× moins cher que GPT-5.5 ou Claude Opus 4.7. Construit sans GPU Nvidia.
- **Testable** : Oui — modèle disponible sur Hugging Face et via API DeepSeek ($0,145/MTok)

## 🔬 Research

### TurboQuant — Réduction de l'overhead KV cache (Google, ICLR 2026)
- **Quoi** : Algorithme combinant PolarQuant (rotation vectorielle) et Quantized Johnson-Lindenstrauss pour comprimer massivement le KV cache sans perte de qualité
- **Lien** : https://www.crescendo.ai/news/latest-ai-news-and-updates
- **Pourquoi c'est important** : Le KV cache est l'un des plus grands goulots d'étranglement pour exécuter des modèles avec de larges fenêtres de contexte. TurboQuant pourrait décaler la course du "plus de paramètres" vers "plus d'efficacité".
- **Testable** : Non — recherche académique (ICLR 2026)

### IA neuro-symbolique : réduction de l'énergie ×100
- **Quoi** : Des chercheurs ont développé une approche hybride combinant réseaux de neurones et raisonnement symbolique, réduisant la consommation d'énergie jusqu'à 100× tout en améliorant la précision
- **Lien** : https://www.sciencedaily.com/releases/2026/04/260405003952.htm
- **Pourquoi c'est important** : Présenté à l'ICRA (International Conference on Robotics and Automation) à Vienne en mai. Pertinent pour l'inférence embarquée sur robots et objets connectés. Ouvre la voie à des agents autonomes à faible consommation.
- **Testable** : Non — recherche académique

### Mollifier Layers — Réseaux de neurones pour équations aux dérivées partielles
- **Quoi** : Chercheurs de l'Université de Pennsylvanie intègrent des fonctions de lissage classiques dans les réseaux de neurones pour résoudre des PDEs inverses avec plus de stabilité
- **Lien** : https://arxiv.org/list/cs.AI/current
- **Pourquoi c'est important** : Adresse un défi de longue date en IA pour la science : simuler des systèmes physiques complexes (mécanique des fluides, météo) avec des réseaux de neurones stables.
- **Testable** : Non — recherche académique
