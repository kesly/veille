# AI Watch — 2026-06-29 Matin

## 🔥 Breaking

### Anthropic Fable 5 & Mythos 5 — Ban gouvernemental partiellement levé
- **Quoi** : Le gouvernement américain autorise désormais un accès limité à Fable 5 et Mythos 5 pour des entreprises et organisations sélectionnées
- **Lien** : https://www.anthropic.com/news/fable-mythos-access
- **Pourquoi c'est important** : Premier ban gouvernemental d'un modèle IA commercial de l'histoire. Lancés le 9 juin, mis hors ligne le 12 juin sur ordre du Commerce Department (Howard Lutnick), citant un jailbreak potentiel lié à l'analyse de code. Anthropic a contesté cette décision ; le gouvernement assouplit maintenant la restriction. Fable 5 est le modèle le plus capable jamais largement déployé par Anthropic : thinking adaptatif permanent, fenêtre contextuelle 1M tokens, 128K tokens de sortie.
- **Testable** : Partiellement — disponible pour les organisations sélectionnées, pas encore grand public

### OpenAI GPT-5.6 Sol — Preview de la prochaine génération
- **Quoi** : OpenAI lance la preview du trio GPT-5.6 : Sol (flagship raisonnement long-horizon), Terra (performances GPT-5.5 à 2× moins cher), Luna (le plus rapide et abordable)
- **Lien** : https://openai.com/index/previewing-gpt-5-6-sol/
- **Pourquoi c'est important** : Sol est conçu pour les workflows agentiques, le coding scientifique et la planification long terme. Aussi soumis à des restrictions d'export américaines. Disponibilité générale annoncée dans les prochaines semaines.
- **Testable** : Non — preview limitée pour l'instant

## 🆕 Nouveaux Outils & Produits

### Vercel eve — Framework open-source pour agents IA
- **Quoi** : Framework Apache-2.0 où chaque agent est un dossier de fichiers (instructions, outils, canaux, schedule) — lancé à Ship London le 17 juin
- **Lien** : https://vercel.com/blog/introducing-eve
- **Pourquoi c'est important** : Vercel décrit eve comme le "Next.js des agents". Déjà utilisé en production pour 100+ agents internes. Déploiement avec `vercel deploy`. TypeScript + Markdown. Exécution durable, sandboxing, approbation humaine, subagents, tracing et evals intégrés.
- **Testable** : Oui — open source, disponible sur https://vercel.com/eve

### Microsoft Scout — Premier "Autopilot" agent d'entreprise
- **Quoi** : Agent autonome avec sa propre identité Entra, connecté nativement à Teams, Outlook, OneDrive, SharePoint — pas besoin d'un prompt pour démarrer
- **Lien** : https://devblogs.microsoft.com/agent-framework/microsoft-agent-framework-at-build-2026/
- **Pourquoi c'est important** : Catégorie inédite d'"Autopilots" qui opèrent en arrière-plan en continu, sans attendre l'utilisateur. Changement de paradigme pour l'agent d'entreprise.
- **Testable** : Non — entreprises sélectionnées uniquement pour l'instant

### Apple Siri AI — Refonte complète de l'assistant
- **Quoi** : Siri entièrement reconstruit sur Apple Intelligence nouvelle génération, avec conscience contextuelle de l'écran et contrôle système complet (8 juin)
- **Lien** : https://www.apple.com
- **Pourquoi c'est important** : Exécution on-device sur hardware compatible. Dictée, rédaction, contrôle d'apps en langage naturel. Premier vrai concurrent pour ChatGPT/Gemini sur mobile.
- **Testable** : Oui — via mise à jour iOS/macOS sur appareils compatibles

### Google NotebookLM — Mise à niveau majeure vers Gemini 3.5
- **Quoi** : NotebookLM passe sur Gemini 3.5, ajoute la découverte agentique de sources, un sandbox cloud pour l'exécution de code, et de nouveaux formats de sortie (8 juin)

- **Pourquoi c'est important** : NotebookLM devient un vrai agent de recherche capable de chercher des sources de manière autonome et d'exécuter du code dans chaque notebook.
- **Testable** : Oui — notebooklm.google.com

### DebugBear MCP Server — Perf web via MCP
- **Quoi** : Nouveau serveur MCP pour accéder à DebugBear (monitoring perf web) depuis Claude, ChatGPT et Cursor (26 juin)
- **Lien** : https://www.debugbear.com/changelog/june-2026-release-notes
