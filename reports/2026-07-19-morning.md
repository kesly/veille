# AI Watch — 2026-07-19 Matin

## 🔥 Breaking
> Releases majeures, annonces qui changent la donne

### GPT-5.6 — Sol, Terra & Luna (OpenAI)
- **Quoi** : OpenAI déploie sa famille GPT-5.6 en trois tiers : Sol (flagship), Terra (milieu de gamme) et Luna (rapide/économique), avec ChatGPT Work pour les tâches autonomes longues
- **Lien** : https://openai.com/index/previewing-gpt-5-6-sol/
- **Pourquoi c'est important** : Sol tourne sur Cerebras à 750 tokens/sec, dépasse Fable 5 sur les benchmarks agentiques. Contexte 1M tokens pour tous les tiers. Tarif Sol : $5/$30 par million tokens, Luna : $1/$6. Cache breakpoints explicites et vie minimale de 30 min
- **Testable** : Oui — disponible sur ChatGPT (plans payants), API OpenAI

### Meta Muse Spark 1.1 — Premier modèle payant de Meta
- **Quoi** : Meta Superintelligence Labs lance Muse Spark 1.1, modèle multimodal agentique avec computer use, contexte 1M tokens, délégation à des sous-agents parallèles
- **Lien** : https://ai.meta.com/blog/introducing-muse-spark-meta-model-api/
- **Pourquoi c'est important** : Premier API payant de Meta ($1.25/$4.25 par million tokens, $20 de crédits gratuits). Bats GPT-5.5 et Opus 4.8 sur les evals agentiques (JobBench, MCP Atlas). Gratuit en mode Thinking dans Meta AI app
- **Testable** : Oui — meta.ai (gratuit, login requis) ou Meta Model API (US uniquement pour l'instant)

### Kimi K3 (Moonshot AI) — Plus grand open-weight au monde
- **Quoi** : Moonshot AI lance Kimi K3, un modèle MoE de 2,8 trillions de paramètres avec un contexte de 1M tokens, ciblant les tâches de coding longue durée et les agents
- **Lien** : https://www.marktechpost.com/2026/07/16/moonshot-ai-releases-kimi-k3-a-2-8-trillion-parameter-open-moe-model-with-kimi-delta-attention-and-1m-context/
- **Pourquoi c'est important** : Premier modèle de classe 3T open-weight — bat Opus 4.8 et GPT-5.5 sur les benchmarks déclarés. Les poids open-source sont promis pour le 27 juillet. Prix : $3/$15 par million tokens (niveau Claude Sonnet)
- **Testable** : Oui — disponible sur Kimi Code et l'app Kimi dès maintenant

### Fable 5 accès gratuit expire ce soir (Anthropic)
- **Quoi** : L'accès gratuit étendu à Claude Fable 5 expire aujourd'hui 19 juillet à 23h59 PT — troisième extension consécutive en réponse au lancement de GPT-5.6 Sol
- **Lien** : https://llm-stats.com/llm-updates
- **Pourquoi c'est important** : Les utilisateurs doivent choisir entre Opus 5, une 4ème extension, ou utiliser des crédits. Signal que la guerre des modèles s'intensifie avec des offres gratuites comme arme de rétention
- **Testable** : Oui — claude.ai

## 🆕 Nouveaux Outils & Produits

### Grok Build open-sourcé (xAI)
- **Quoi** : xAI publie l'intégralité du code source de Grok Build sous licence Apache 2.0 (844 530 lignes de Rust). Ajout du STT via `/voice` ou `Ctrl+Space`

- **Testable** : Oui — open source, gratuit

### Meta Muse Image

### MCP Spec Release Candidate
- **Quoi** : Le release candidate de la spécification MCP est disponible, finale prévue le 28 juillet 2026 — plus grande révision depuis le lancement
- **Lien** : https://blog.modelcontextprotocol.io/posts/2026-07-28-release-candidate/
- **Pourquoi c'est important** : Core stateless scalable sur HTTP standard, MCP Apps (UIs server-rendues), extension Tasks pour les travaux longs, OAuth/OpenID Connect aligné. L'écosystème compte désormais 5 800+ serveurs communautaires

### Safari MCP Server (Apple)
- **Quoi** : Apple a lancé le 1er juillet le premier MCP serveur officiel d'un browser majeur, exposant 16 outils d'accès au Web Inspector de Safari
- **Lien** : https://developer.chrome.com/blog/chrome-devtools-mcp
- **Pourquoi c'est important** : Les agents IA peuvent désormais déboguer des apps web directement dans Safari. Chrome DevTools MCP Server également en public preview

### Sim — Workspace open-source pour agents IA
- **Quoi** : Workspace open-source pour construire et gérer des agents et workflows IA (595 upvotes sur Product Hunt le 9 juillet)
- **Lien** : https://startupcorners.com/digest/product-digest-2026-07-09

## 🔬 Research

### J-lens — Interprétabilité par Jacobien (Anthropic)
- **Quoi** : Technique d'interprétabilité basée sur la matrice Jacobienne qui identifie un sous-espace interne (~25 concepts actifs, <10% de la variance d'activation) se comportant comme le "workspace global" de la théorie de la conscience
- **Lien** : https://aiweekly.co/ai-news-today/anthropic-news
- **Pourquoi c'est important** : Premier indice structurel d'un mécanisme analogue à la conscience dans les LLMs. Ouvre la voie à une interprétabilité mécanistique plus fine
- **Testable** : Non — recherche fondamentale

### Selective Activation Sparsity (ICML 2026)
- **Quoi** : Méthode d'entraînement qui force les modèles à n'activer que les paramètres les plus pertinents par tâche — performances comparables à des modèles 3x plus grands sur les benchmarks de raisonnement
- **Lien** : https://skycrumbs.com/blog/ai-research-july-2026
- **Pourquoi c'est important** : Si scalable, réduction drastique des coûts d'inférence et possibilité de faire tourner des modèles capables sur mobile/laptop
- **Testable** : Non — résultats de recherche

### Prospective Credit Assignment (DeepMind, NeurIPS 2026)
- **Quoi** : Nouvelle approche d'entraînement pour apprendre aux modèles à anticiper l'impact de décisions actuelles sur des résultats plusieurs étapes à l'avance
- **Lien** : https://skycrumbs.com/blog/ai-research-july-2026
- **Pourquoi c'est important** : Approche pour résoudre le problème de planification à long terme — clé pour les agents autonomes complexes
- **Testable** : Non — soumis à NeurIPS 2026
