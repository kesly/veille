**Fondateur :** donghaxkim (GitHub)

- **Target :** Devs React/Next.js/Vite qui utilisent Tailwind
- **Pricing :** Open source (pour l'instant)

- CLI Node.js, proxy local, overlay injecté
- Support: Next.js, Vite, CRA
- Tailwind CSS pour l'éditeur de propriétés

### Psychologie du succès
- ✅ ROI immédiat (gain de temps visible)
- ✅ Simplicité (npx react-rewrite et c'est parti)
- ✅ Communauté dev (OSS = trust)
- **JTBD :** "Quand j'ajuste du styling React, je veux voir et éditer visuellement, pour shipper plus vite sans alt-tab"
- **Aha moment :** Premier clic sur un élément → édition live → fichier source mis à jour

- **Viral loop :** Dev essaie → tweete "wow" → plus de devs

### Opportunités de réplication

## 📈 Tendances Émergentes

### 1. 🔧 "Vibe Coding" devient une catégorie à part entière
Product Hunt a créé une catégorie dédiée. Fractal, Crossnode, Google AI Studio 2.0 — tous convergent vers "décris ce que tu veux, l'agent code pour toi". Les outils *autour* du vibe coding (editing, testing, context) sont les vraies opportunités.

### 2. 🧠 Context-as-Infrastructure
Pensieve, Littlebird, ClawMem — la course est à "qui donne le meilleur contexte aux AI agents". Le RAG basique ne suffit plus. Knowledge graphs, mémoire persistante, context stacks = le prochain layer d'infra.

### 3. 🎪 Viral Novelty Apps = Marketing Masterclass
SlapMac prouve qu'un concept absurde + exécution rapide + distribution multi-plateforme = $5K en 3 jours sans un centime de pub. Pattern réplicable.

### 4. 🏗️ OSS Dev Tools → SaaS Pipeline
React-Rewrite, ChameleonDB, Manifold — le playbook est clair: ship OSS, get adoption, add pro tier. Les dev tools OSS sont le nouveau "freemium landing page".
