- **USP:** Contrairement à Rewind (screenshots), Littlebird "comprend" le contexte textuel → pas besoin de prompts
- **Target:** Knowledge workers, devs, managers — $15-25/mois estimé
- **TAM:** ~500M+ (productivité AI globale)

### 3️⃣ STACK TECHNIQUE
- Desktop app (macOS/Windows)
- Screen reading (OCR/text extraction en local)
- Stockage texte local (privacy-first)
- LLM pour search/query sur le contexte capturé
- Intégrations optionnelles (apps favorites)

### 4️⃣ PSYCHOLOGIE DU SUCCÈS
- [x] **ROI immédiat** — "ne plus re-expliquer le contexte"
- [x] **Simplicité** — install & forget, zéro config
- [x] **FOMO** — "les autres ont déjà une mémoire AI, pas toi?"
- [x] **Autorité** — TechCrunch + $11M = crédibilité
- **JTBD:** Quand je suis en réunion/travail et que l'AI me demande du contexte, je veux que tout soit déjà capturé, pour ne jamais ré-expliquer
- **Aha moment:** Première query qui retrouve un détail d'il y a 3 jours sans avoir rien noté

### 5️⃣ GO-TO-MARKET
- **Canaux:** TechCrunch PR → Product Hunt → Twitter/HN organic
- **Stratégie:** Freemium + premium pour stockage/features avancées
- **Viral loop:** "Comment tu savais ça?" → "Littlebird capture tout"

### 6️⃣ OPPORTUNITÉS DE RÉPLICATION
- **Score complexité:** 7/10 (screen reading + LLM + privacy = technique)
- **Verticaux adjacents:** Version pour devs uniquement (code context), version pour sales (call memory)
- **Quick wins:** Intégration native avec Claude Code / Cursor → "coding memory"
- **Notre angle:** Version verticale FR pour consultants/comptables (mémoire des dossiers clients)
- **Estimation:** 8-12 semaines, 1 dev senior

**🎯 Verdict:** ⭐⭐⭐⭐⭐ (5/5)
**⏱️ Time-to-replicate:** 10 semaines (version verticale)
