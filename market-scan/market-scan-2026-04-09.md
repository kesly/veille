- Sensor Tower : +56% YoY toutes plateformes
- **Signal :** La barre d'entrée pour créer des apps est au sol. La DISTRIBUTION et le MARKETING deviennent le vrai moat, pas le code.

### 3. 🤖 Agents > Copilots
- Cursor 3 : d'un copilot → fleet d'agents autonomes
- Audos/Otto : agent qui build + lance + vend
- Lessie AI, Cockpit AI : agents de prospection/outreach
- **Signal :** Le marché passe de "AI qui aide" à "AI qui fait". Les wrappers autour de LLMs meurent. Les agents verticaux avec workflow complet survivent.

### 4. 🏰 Le Moat se déplace
- Code = commodity (vibe coding)
- Data + distribution + workflow = nouveau moat
- Les big tech (Meta Muse) poussent vers l'assistant personnel unique
- **Signal :** Construire des agents avec accès à des données propriétaires (ex: données comptables FR, menus restaurants) = le play gagnant.
