## 📰 Use Cases & Actualités (dernières 24h)

### 1. 🏢 Nvidia, Anthropic, Perplexity, Snowflake accélèrent leurs agents en réponse à OpenClaw — **8/10**
- **Source:** Axios
- **Quoi:** Les Big Tech lancent des alternatives/compléments à OpenClaw pour les entreprises, validant le modèle d'agents autonomes 24/7
- **Impact:** OpenClaw est devenu le benchmark de facto pour les agents AI personnels — les géants construisent autour ou en réponse
- **Pourquoi c'est important:** Signal fort que le marché des agents autonomes est réel et en pleine explosion. OpenClaw a créé la catégorie.

### 2. ⚡ OpenClaw v2026.3.22 — Sessions 48h + sécurité massive — **8/10**
- **Source:** Efficienist, GitHub, Reddit r/openclaw
- **Quoi:** Timeout par défaut passe de 10 min à 48h. Suppression complète des noms legacy (CLAWDBOT_*, MOLTBOT_*). MiniMax M2.7 comme modèle par défaut. Claude via Vertex AI natif. Plugins web (Exa, Tavily, Firecrawl) bundlés. Android: SMS search, call logs, dark theme.
- **Sécurité:** Fix faille Windows SMB, Unicode invisible dans exec prompts, auth webhook/pairing renforcée
- **Bug notable:** Control UI cassé (#52808) — `dist/control-ui/` manquant du package npm
- **Pourquoi c'est important:** La release la plus riche en sécurité jamais livrée. Les sessions 48h changent fondamentalement ce que les agents peuvent accomplir en continu.

### 3. 🎨 Hitem3D — Skill ClawHub pour génération Image → 3D — **7/10**
- **Source:** PRNewswire (Math Magic, Singapore)
- **Quoi:** Hitem3D publié comme Skill callable sur ClawHub. Supporte GLB, OBJ, STL, FBX, USDZ. Modes geometry-only ou texturé.
- **Pourquoi c'est important:** Premier Skill 3D sérieux sur ClawHub. Ouvre la porte à des workflows agent qui génèrent des assets 3D depuis une simple image — prototypage, e-commerce, jeux.

### 4. 🐇 Rabbit R1 pousse son intégration OpenClaw — **7/10**
- **Source:** Yanko Design
- **Quoi:** Rabbit R1 approfondit son intégration OpenClaw (alpha depuis jan 2026). Push-to-talk → OpenClaw exécute via le gateway de l'utilisateur.
- **Pourquoi c'est important:** Premier hardware grand public à intégrer OpenClaw nativement. Si Rabbit réussit, c'est un modèle pour d'autres devices (montres, enceintes, etc.)

### 5. 🇨🇳 Chine restreint OpenClaw dans les agences d'État — **6/10**
- **Source:** Wikipedia (citant rapports mars 2026)
- **Quoi:** Entreprises d'État et agences gouvernementales chinoises interdites d'utiliser OpenClaw sur les machines de bureau
- **Pourquoi c'est important:** Signal géopolitique. Montre qu'OpenClaw est perçu comme assez puissant pour être un risque de sécurité nationale. Paradoxe avec l'intégration WeChat de Tencent (voir veille hier).

### 6. 📝 Substack critique : "2 Reasons I Turned Off My OpenClaw" — **5/10**
- **Source:** GAI Insights (Substack)
- **Quoi:** Critique de l'instabilité et de la complexité technique. "Strong potential, but still unstable."
- **Pourquoi c'est important:** Feedback terrain réaliste. L'UX reste la barrière #1 à l'adoption mainstream.

### 7. 🔧 Communauté chinoise : tutoriels sandbox bypass — **4/10**
- **Source:** LINUX DO (forum chinois)
- **Quoi:** Tutoriels détaillés pour contourner le sandbox d'OpenClaw et donner full permissions à l'agent
- **Pourquoi c'est important:** Signal d'adoption massive en Chine, mais aussi risque sécurité. La communauté hacke le produit pour l'utiliser "sans filet."
