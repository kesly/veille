## 🔥 Use Cases & Actualités (dernières 24h)

### 1. Anthropic impose le paiement API pour les agents tiers (OpenClaw inclus)
**Score: 9/10** 🔴
- **Source:** CNET (12 avr, ~20h UTC)
- **Quoi:** Anthropic annonce officiellement que les abonnés Claude ($20/mois) devront désormais payer via l'API ou un "pay-as-you-go" séparé pour utiliser des agents tiers comme OpenClaw. Citation virale d'Aakash Gupta : *"The $20/month all-you-can-eat buffet just closed."*
- **Contexte:** Suite directe de l'affaire Steinberger (ban temporaire J-4). Anthropic pousse aussi ses propres features (Computer Use) pour concurrencer OpenClaw directement.
- **Impact Kyle:** Changement majeur de pricing. Si on utilise Anthropic via OpenClaw, il faudra budgéter l'API. Confirme la tendance : les labs veulent reprendre le contrôle du trafic agent.

### 2. Hermes Agent (Nous Research) — premier vrai rival d'OpenClaw
**Score: 8/10** 🟢
- **Source:** TuringPost (12 avr)
- **Quoi:** Article de fond comparant Hermes Agent vs OpenClaw. Hermes est un agent self-hosted, model-agnostic, avec un système de mémoire procédurale (l'agent transforme ses workflows réussis en skills réutilisables). Différence clé : Hermes = self-improving loop vs OpenClaw = control-plane + human-authored skills.
- **Points forts Hermes:** Mémoire layered (persistent notes, session history, procedural knowledge), safer-by-default, accumulation de compétences automatique.
- **Impact Kyle:** Premier concurrent sérieux. L'approche "self-improving" est intéressante pour nos propres workflows. À surveiller de près.
