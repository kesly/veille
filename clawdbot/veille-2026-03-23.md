## 📰 Use Cases & Actualités (dernières 24h)

### 1. 🇨🇳 Tencent intègre WeChat × OpenClaw — **9/10**
- **Source:** Reuters
- **Quoi:** Tencent lance "ClawBot" comme contact WeChat, connectant 1+ milliard d'utilisateurs à OpenClaw
- **Impact:** Adoption massive potentielle en Chine via l'app la plus populaire du pays
- **Pourquoi c'est important:** Premier déploiement OpenClaw à l'échelle d'un super-app. Signal que les Big Tech asiatiques adoptent le framework plutôt que de recréer.

### 2. 🏗️ Pattern n8n + OpenClaw : orchestration multi-agents — **7/10**
- **Source:** Kollox.com (article technique détaillé)
- **Quoi:** Architecture émergente utilisant n8n comme couche d'orchestration/guardrails pour des agents OpenClaw spécialisés (researcher, critic, writer)
- **Pattern clé:** n8n gère state, tool provisioning, guardrails et human-in-the-loop ; OpenClaw agents exécutent les tâches autonomes
- **Pourquoi c'est important:** Résout le problème de gouvernance des agents autonomes en production. Architecture reproductible pour les entreprises.

### 3. 🔐 Claude OAuth supprimé pour les apps tierces — **7/10**
- **Source:** DaveSwift.com, commit GitHub OpenCode
- **Quoi:** Anthropic retire le support OAuth Claude des apps tierces comme OpenClaw
- **Impact:** Fix temporaire disponible, mais changement confirmé à venir
- **Pourquoi c'est important:** Anthropic serre la vis sur l'accès tiers — pousse vers Claude Code Channels (leur alternative maison). Risque pour les utilisateurs OpenClaw qui dépendent de Claude via OAuth.

### 4. 📰 Forbes critique l'UX d'OpenClaw — **6/10**
- **Source:** Forbes (Paul Baier)
- **Quoi:** Article "2 Reasons I Turned Off My OpenClaw" — critique le produit comme "raw, untested code" pas adapté aux non-devs
- **Impact:** Premier article critique majeur dans la presse mainstream
- **Pourquoi c'est important:** Signal de maturité : OpenClaw est assez gros pour recevoir des critiques mainstream. Valide le besoin d'UX simplifiée.

### 5. 🐛 Bug plugin WeChat communautaire (openclaw-weixin) — **5/10**
- **Source:** GitHub Issue #52341
- **Quoi:** Plugin `openclaw-weixin` ne charge pas — erreur `Cannot find module 'openclaw/plugin-sdk'`
- **Impact:** Bloque les utilisateurs chinois qui veulent connecter WeChat en attendant l'intégration officielle Tencent
- **Pourquoi c'est important:** Montre la tension entre écosystème communautaire et API officielle.

### 6. 🇨🇳 Chine restreint OpenClaw dans les agences d'État — **6/10**
- **Source:** Wikipedia (citation presse chinoise)
- **Quoi:** Le gouvernement chinois interdit OpenClaw dans les agences d'État et entreprises publiques pour raisons de sécurité
- **Impact:** Paradoxe : Tencent l'adopte côté privé, l'État le bloque côté public
- **Pourquoi c'est important:** Pattern classique des tensions tech US-Chine. N'empêche pas l'adoption massive côté consommateur/entreprise privée.

### 7. 🔒 Campagne phishing GitHub (suite) — **5/10**
- **Source:** WEEX Crypto News
- **Quoi:** Campagne sophistiquée ciblant les contributeurs OpenClaw sur GitHub — comptes attaquants supprimés
- **Impact:** Vigilance accrue requise pour les développeurs du projet
- **Pourquoi c'est important:** Continuation de la campagne détectée hier. OpenClaw est une cible de valeur.

### 8. 🤖 Xiaomi MiMo-V2-Pro : API gratuite pour OpenClaw — **6/10**
- **Source:** DevFlokers
- **Quoi:** Xiaomi offre un accès API gratuit à son modèle MiMo-V2-Pro spécifiquement pour les intégrations OpenClaw
- **Impact:** Nouveau modèle gratuit disponible pour les utilisateurs OpenClaw
- **Pourquoi c'est important:** Les fabricants de modèles courtisent l'écosystème OpenClaw — preuve de son influence comme plateforme de distribution.
