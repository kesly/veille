# Veille ClawdBot/OpenClaw — 2026-03-04

## 📦 Release: OpenClaw v2026.3.2 (3 mars 2026)

### Changements majeurs
1. **SecretRef mature** — 64 cibles de credentials supportées, fail-fast sur refs non résolues, `openclaw secrets audit`
2. **Outil PDF natif** — support Anthropic & Google, limites configurables, extraction fallback
3. **sendPayload unifié** — Discord, Slack, WhatsApp, Zalo, Telegram partagent un adaptateur commun avec chunking et multi-media
4. **Telegram streaming** — défaut `partial` (live typing) pour nouvelles installs
5. **Profil tools par défaut** — nouvelles installs = `messaging` au lieu de `full` (sécurité)

## 🔥 Use Cases Innovants (dernières 24h)

### 1. Agent PDF B2B — Analyse contrats & RFP (8/10)
**Source:** Reddit r/OpenClawUseCases, article ucstrategies.com
Avec le nouvel outil PDF natif, des utilisateurs analysent des contrats pour extraire obligations par partie, comparent des méthodologies de papers de recherche, et génèrent des réponses structurées à des RFP. Use case B2B immédiatement monétisable.

### 2. Home Assistant + OpenClaw — Smart Home IA (7/10)
**Source:** markaicode.com (guide publié hier)
Guide complet pour intégrer OpenClaw avec Home Assistant en 20 min. Contrôle vocal via messaging apps, ajustement chauffage selon météo, automatisation complète du foyer. L'intersection IoT + agent IA se concrétise.

### 3. Inbox Automation à grande échelle — 4000 emails en 2 jours (7/10)
**Source:** ucstrategies.com (thread Matt Van Horn)
Agent OpenClaw qui traite des milliers d'emails: désabonnements spam, catégorisation par urgence, brouillons de réponses. Un utilisateur a nettoyé 4000+ emails en laissant l'agent tourner la nuit. Pattern reproductible.

### 4. Restaurant Tipping Automation — Watch & Learn (7/10)
**Source:** ucstrategies.com
OpenClaw observe un screen recording d'un process de tips en restaurant et le reproduit automatiquement. Paradigme "show don't tell" — l'agent apprend par observation plutôt que par instructions écrites. Pertinent pour notre projet RestoAI.

### 5. School WhatsApp Monitor + Face Recognition (6/10)
**Source:** ucstrategies.com
Agent qui surveille un groupe WhatsApp scolaire, filtre le bruit, fait de la reconnaissance faciale sur les photos, et envoie un digest quotidien aux parents. Use case niche mais impressionnant techniquement.

### 6. Trading Crypto avec Sentiment Social (6/10)
**Source:** ucstrategies.com
Agents multi-jours: calcul taille positions, stop-loss, alertes, exécution auto sur exchanges. Monitoring du sentiment social + API exchanges + notifications sur seuils. Pattern risqué mais demandé.

### 7. Alibaba Cloud intègre OpenClaw nativement (5/10)
**Source:** help.aliyun.com
Alibaba Cloud (Aliyun) propose des guides officiels pour déployer OpenClaw avec DingTalk (messagerie chinoise) et Qwen3. Signe de l'adoption enterprise en Chine.

### 8. /last30days Skill — Research en 30 secondes (5/10)
**Source:** Matt Van Horn (@mvanhorn) sur X
Skill Claude Code qui scanne Reddit, X et le web sur les 30 derniers jours pour un topic donné. Retourne prompt patterns + releases + workflows actuels. Meta-outil de veille.
